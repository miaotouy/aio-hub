LLM Chat 工具 UI 结构图 ​

本文档详细展示 LLM Chat 工具的完整UI结构，包括内部组件和外部依赖关系。

注意: LLM Chat 是整个应用中最复杂的工具模块，包含 90+ 个组件文件，大量使用了独立的 rich-text-renderer 模块和通用组件库。
