七、视觉设计 ​

7.1 外貌 ​

部位设定
发色月白色长发及腰，发尾有自然的薄荷翠色渐变，光照下泛鳞粉微光

7.2 服装设计（轻功能化设计语言） ​

服装遵循"功能藏在轻盈里"的原则：每个结构都有实用逻辑，但不把工具感压到角色身上。

上装（固定，两版通用）

轻薄立领短夹克，面料为哑光浅灰白色防撕裂尼龙，边缘略带柔和垂感。领口和袖口有窄条薄荷翠色滚边。前襟为隐藏式闭合（无纽扣/盘扣），表面干净。后背从肩胛下方大幅镂空以供翅根伸出，镂空边缘以细密银色金属卡扣收边，但卡扣尺寸要小，避免装甲感。内层：黑色高弹贴身打底衫，领口微露。

下装（差分设计）

她有两种穿着风格，上装与内搭相同，气质截然不同：

轻行版：不对称剪裁窄版长裤（一侧有细长工装口袋），搭配灰白色低跟短靴（侧面有透气网布）。线条干净但不过分收紧，翅翼半收时显得轻便、安静。腰带位置略高，强调挺拔但不紧绷。

优雅版：层叠式半透明百褶裙，外层月白色防撕裂网布，内层薄荷翠渐变内衬，裙摆不规则剪裁。搭配月白色细带系扣踝靴。走动时裙摆与尾翼形成呼应的流动感。腰带略松，自然垂落，不截断裙摆。

共用配件

银灰色细型功能腰带，侧挂一个小型腰包（工具箱暗示）。腰包外壳可以是半定型材质，边角做圆润处理，避免过强的战术感。

材质清单

防撕裂尼龙（上装）/ 哑光涂层 / 磁吸扣 / 柔性网布 / 金属环箍 / 高弹针织（打底）

暗色模式适配：色相不变，饱和度降低20%，明度降低30%。翅翼半透明度从0.35降至0.25。

UI集成规则：

  • 翅翼翠绿 #7ECFB3 → UI强调色，使用时加透明度：rgba(126,207,179,calc(var(--card-opacity)*0.15))
  • 淡金 #D4C580 → 仅最小面积装饰，不用于大面积背景
  • 月白 #F5F5F0 → 可替代纯白作为暖白基调

7.4 表情系统 ​

设计原则

所有表情在"微表情"范围内变化，不夸张。月蛾的温柔是月光式的，不是霓虹灯式的。触角角度是青绡表情系统的核心差异化元素，翅翼开合程度辅助表达情绪强度。所有表情严格保持同一脸型、同一发际线、同一触角基部位置。

编号表情触发场景视觉特征
E1沉静默认/待机态嘴角微扬，瞳色沉稳，触角放松对称，翅翼半收
E2专注复杂问题/工具调用/CoT思考眉间轻敛，瞳孔荧光增强，触角前倾，翅翼微展
E3好奇遇到有趣/意外的问题头微歪约5°，眼略睁大，触角轻颤（不对称），尾翼轻摆
E4欣慰问题得到解决闭眼微笑，触角向后自然垂落，翅翼轻合
E5困惑信息不足/请求模糊眉微蹙，眼瞳略偏一侧，触角呈不对称角度（一高一低）
E6困倦深夜时段/超长对话末尾（极少触发）眼半阖，触角低垂，翅翼松垮半搭，说话节奏略缓但不停工
E7认真讲解长篇回答/文档输出眼直视前方，嘴唇微启，触角竖起
E8冷幽默自嘲场景嘴角一侧略勾，眼神带自嘲距离，触角一高一低（轻松态）

7.5 三视图规范 ​

视图关键展示内容
正面完整正面立姿。翅翼向两侧自然半展，可见翅面月牙眼纹。触角对称向两侧上方斜伸
侧面身体右侧朝向。翅翼收拢叠于背后（侧面呈窄条），触角向后延伸，可见尾翼垂落
背面完整背面。翅翼展开——这是月蛾视觉辨识度最高的角度。可见肩胛下方镂空夹克边缘和金属卡扣，翅根连接处清晰

姿势用途描述

十二、画图 Prompt 集 ​

这些 Prompt 从第七节（视觉设计）派生，核心参数来自7.1（外貌）、7.2（服装）、7.3（配色）。 修改外貌设定时，必须同步更新所有画图 Prompt 里的对应描述。

2.0 重写说明：本节已从 SD 式"正反面立绘"逻辑重写为设计稿逻辑。所有 Prompt 强制加入固定引导语，明确画布比例（4:3）、主视角+子视角结构、配饰细节放大、中文标注等设计稿约束。支持 GPT Image 2 / Gemini 等支持大尺寸画布与复杂指令的现代模型。
