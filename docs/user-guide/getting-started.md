# 快速开始

## 概述

**AIO Hub** 是一款面向 AI 创作者和开发者的桌面工作站，把你日常所需的 AI 对话、内容创作、文档处理等工具整合到一个应用中。你不再需要在多个网页和软件之间来回切换——只需打开 AIO Hub，就能一站式完成从灵感获取到成品输出的全流程。

### 核心功能

- **💬 LLM 智能对话** — 与 ChatGPT、Claude、Gemini 等多种 AI 模型对话，支持多轮上下文跟踪、智能体切换和长文本压缩
- **🧩 30+ 实用工具** — 从图片 OCR 文字识别、视频转写、Git 仓库分析到代码格式化、JSON 格式化，覆盖日常开发与创作的方方面面
- **🪟 自由窗口管理** — 任何工具模块的面板都能设置为独立窗口，多任务并行不打架
- **🎨 极致视觉体验** — 支持 Windows Mica/Acrylic 毛玻璃特效和动态壁纸，每次打开都是享受
- **🔌 开放生态** — 原生支持 OpenAI、Anthropic、Gemini 等主流服务商，兼容 Ollama 本地模型，还有 JavaScript/Rust 插件系统等你来扩展

### 2. 添加你的第一个 LLM 服务

要跟 AI 对话，你需要先告诉 AIO Hub 用哪个模型。

1. 点击窗口右上角的 **⚙️ 设置** 图标
2. 在设置页面中，选择 **LLM AI 服务配置**
3. 点击左侧面板的 **添加服务** 按钮，在弹出的对话框中选择一个服务商预设（如 OpenAI、DeepSeek），或点击右上角"从空白创建"手动配置
4. 在右侧表单中填入你的 API Key 和服务商地址，配置会自动保存

![添加 LLM 服务](TODO: 截图-点击添加服务后的配置表单)

> 💡 **第一次用？** 推荐使用 [SiliconFlow](https://siliconflow.cn) 或 [DeepSeek](https://platform.deepseek.com) 注册并获取 API Key，两个平台都有免费额度。

添加完成后，你可以在左侧列表里看到刚创建的服务配置，右侧可继续编辑参数。

### 3. 开启第一次对话

1. 点击侧边栏顶部的 **🏠 主页** 图标（或默认启动页），在工具网格中找到并点击 **💬 LLM 聊天** 卡片。
2. **选择或创建智能体**：
   - 第一次进入时，你会看到左侧的 **🔮 智能体** 列表。
   - 点击底部的 **添加智能体** 按钮，选择一个预设（如"全能助手"）或从空白创建。
   - 在弹出的对话框中，确保选择了你刚才配置的服务和模型，然后点击确定。
3. **新建会话**：
   - 选中智能体后，点击右侧侧边栏顶部的 **➕** 按钮（新建对话）。
4. **开始聊天**：
   - 在中间底部的输入框里打字，点击发送按钮或按下 `Ctrl + Enter` 发送消息。

## 功能详解

### 主界面布局

AIO Hub 的主界面分为三个核心区域：

```
┌────────────────────────────────────────────────────┐
│                    标题栏                           │
├─────────────┬──────────────────────────────────────┤
│             │                                      │
│  侧边栏     │         主内容区域                     │
│  (工具导航)  │     (聊天/工具页面)                   │
│             │                                      │
│             │                                      │
└─────────────┴──────────────────────────────────────┘
```

![界面布局标注](TODO: 截图-用箭头/框线标注出三个区域)

- **标题栏**：顶部区域，显示应用名称和窗口控制按钮。AIO Hub 使用自定义标题栏，支持更优雅的视觉风格。
- **侧边栏**：左侧竖条，作为**运行时标签栏**。它仅显示你当前**已打开**的工具。你可以通过侧边栏在多个任务间快速切换。要打开新工具，请点击顶部的 **🏠 主页** 图标。
- **主内容区域**：显示当前选中的工具页面。

### 💬 LLM 聊天面板

聊天是 AIO Hub 的旗舰功能，不仅是一个简单的对话窗口。

#### 关键参数说明

| 参数           | 位置              | 说明                                                                 |
| :------------- | :---------------- | :------------------------------------------------------------------- |
| **模型选择**   | 聊天窗口右上角    | 选择当前要使用的 AI 模型。每个 LLM 服务下可以配置多个模型            |
| **智能体选择** | 模型选择器旁边    | 切换不同的 AI "人设"。你可以创建多个智能体，每个有独立的设定和知识库 |
| **上下文长度** | 聊天窗口顶部/底部 | 显示当前对话消耗的 Token 数量。蓝色条表示已用比例                    |
| **输入模式**   | 输入框左侧图标    | 切换纯文本 / 带格式 / 附件模式                                       |

![聊天面板参数标注](TODO: 截图-在聊天界面上标注出上述参数位置)

#### 常用操作

- **发送消息**：输入文字后按 `Ctrl + Enter`
- **换行**：直接按 `Enter`
- **上传附件**：点击输入框左侧的 📎 图标，或直接拖拽文件到输入区域

## 高级用法

掌握了基础操作后，这些高阶技巧能让你的效率更上一层楼。

### 🪟 组件分离拖拽

AIO Hub 的多数面板都支持拖拽为独立窗口——这是它最独特的功能之一。

**场景：边聊天边看文档**

1. 打开 LLM 聊天面板
2. 把鼠标移到聊天面板的标题栏
3. 按住标题栏拖拽到主窗口之外

![拖拽分离](TODO: 截图-聊天面板被拖拽出主窗口的瞬间)

聊天面板会变成一个独立的悬浮窗口，你可以把它拖到另一个屏幕。这时再在主窗口里打开其他工具——两个窗口互不干扰，状态同步更新。

**可分离的面板**：聊天输入框、对话区域、媒体生成器、JSON 格式化等几乎所有工具面板都支持。

### ⌨️ 快捷键参考

| 快捷键         | 功能                       |
| :------------- | :------------------------- |
| `Ctrl + Enter` | 聊天发送消息               |
| `Esc`          | 取消正在进行的拖拽分离操作 |

> 💡 **提示**：在代码编辑器（如 JSON 格式化、文本对比）中，你可以使用 Monaco Editor 内置的快捷键，例如 `Ctrl + F` 进行搜索。

### 🎨 深度主题定制

如果默认的外观还不能让你满意，可以试试：

1. **动态壁纸**：在设置中开启"动态壁纸"，选择内置壁纸或添加自己的图片/视频
2. **CSS 编辑器**：在设置 → 外观 → CSS 编辑器中直接写 CSS 覆盖样式
3. **样式宏**：利用 CSS 变量快速调整全局配色（参考 [CSS 变量宏指南](../user-guide/tools/llm-chat/css-variables-guide)）

### 🔌 使用插件扩展功能

AIO Hub 支持三种插件类型：

- **JavaScript 插件**：轻量级 UI 扩展，即写即用。放在 `plugins/` 目录下即可自动加载
- **Native 插件 (Rust)**：高性能后端扩展，通过 DLL 动态加载
- **Sidecar 插件**：任意语言编写的独立进程插件

> 插件开发教程请参考 [插件开发指南](../guide/plugin-development-guide.md)。

每个模型看到的都是完整的对话历史，切换无缝衔接。

## 常见问题

### Q1：为什么发送消息后没有回复？

可能的原因和解决方法：

1. **LLM 服务配置有误** — 去设置中点击"测试连接"确认 API Key 和地址填写正确
2. **网络问题** — 如果使用本地模型（Ollama），确认服务是否在运行；如果使用云端 API，确认网络可以访问
3. **模型选择不对** — 确认聊天窗口右上角已选中了正确的模型

### Q2：如何添加自己的 API Key？

进入 **设置 → LLM 服务**，点击 **添加服务**：

- **服务商**：选择预设的 OpenAI / Anthropic / Gemini / DeepSeek / SiliconFlow 等
- **API Key**：从对应平台复制并粘贴
- **模型列表**：系统会自动拉取可用模型，也可以手动添加

如果列表中找不到你的服务商，可以选择"自定义"，手动填入 API 地址。

### Q3：窗口分离后怎么恢复？

- **拖拽回去**：把分离窗口的标题栏拖回主窗口区域，它会自动吸附合并
- **关闭分离窗口**：点击分离窗口的关闭按钮，它会自动回到主窗口中原来的位置

> 所有分离窗口的位置和大小都会被自动记住，下次启动应用时会自动恢复。

### Q4：应用卡顿或崩溃怎么办？

1. **更新到最新版本**：预览版修复了许多稳定性问题
2. **清理缓存**：关闭应用后，删除 `%APPDATA%\com.mty.aiohub\cache` 目录（Windows）
3. **查看日志**：日志文件位于 `%APPDATA%\com.mty.aiohub\logs\`，可用于排查具体原因
4. **反馈问题**：在 GitHub Issues 中提交 Bug 报告，附上日志文件

### Q5：如何卸载 AIO Hub？

- **Windows**：在"设置 → 应用"中找到 AIO Hub 并卸载，或使用安装目录下的卸载程序
- **macOS**：将应用从"应用程序"文件夹拖入废纸篓
- **Linux**：如果是通过包管理器安装，使用对应的卸载命令；如果是 AppImage，直接删除文件即可

> 卸载后，配置文件和数据（`%APPDATA%\com.mty.aiohub`）默认会保留，如果需要彻底清除，请手动删除该目录。
