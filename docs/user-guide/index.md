# 用户手册

## 关于 AIO Hub

AIO Hub 是一款**跨平台桌面 AI 工作站**，将 LLM 智能对话、30+ 专业工具和开放插件系统整合到一个统一的界面中。无论你是 AI 爱好者、开发者还是创作者，都能在这里找到提升效率的利器。

:::tip 快速了解
如果你是第一次接触 AIO Hub，建议先阅读 [项目概览](./project-overview.md) 了解它能为你做什么。
:::
