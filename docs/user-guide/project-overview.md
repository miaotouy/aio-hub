# 项目概览

欢迎来到 **AIO Hub** —— 你的专属 AI 创作与开发工作站。这份概览将带你快速了解这个工具能为你做什么，以及它适合什么样的你。

---

## 概述

### AIO Hub 是什么？

AIO Hub 是一个**跨平台的桌面应用程序**，它把 LLM（大语言模型）对话、30+ 专业工具和强大的插件系统整合到一个统一的工作台中。简单来说，它是一个你可以**自由组装**的工具箱。

### 它解决什么问题？

- **告别碎片化**：AI 对话、代码工具、文件处理、正则匹配、OCR 识别……这些本来需要开五六个窗口的事情，现在一个窗口搞定。
- **告别黑盒 Prompt**：想看清楚你发给 AI 的每个 Token 到底长什么样？AIO Hub 提供全链路透明化，让你完全掌控 Prompt 的每个细节。
- **告别死板的布局**：不习惯固定的左右分栏？把聊天窗口拖出来变成独立悬浮窗，想放哪放哪。

### 适合谁用？

| 如果你是……                   | 你会喜欢它的……                                                            |
| ---------------------------- | ------------------------------------------------------------------------- |
| **AI 爱好者 / 角色扮演玩家** | 强大的 LLM 对话系统、深度上下文控制、SillyTavern 世界书兼容、非线性对话树 |
| **开发者**                   | 文本差异对比、正则批量处理、Git 仓库分析、代码格式化、LLM 请求检查器      |
| **创作者 / 写作者**          | 智能 OCR、音视频转写、富文本渲染、Markdown 实时预览、FFmpeg 媒体处理      |
| **效率工具控**               | 自由窗口管理、插件系统、主题定制、动态壁纸、30+ 工具一站整合              |

## 核心特性

### 💬 LLM 智能对话大厅

不止是聊天。AIO Hub 的对话工作区支持**非线性对话图谱**（对话历史以树状结构展示，你可以随时在分支间切换）、**上下文压缩**（长对话自动智能摘要，不浪费 Token）、**60+ 动态宏**（插入时间/系统信息/角色变量等），以及**全链路 Prompt 可视化**——发送前的原始 Prompt 长什么样，一清二楚。

> 如果你玩过 SillyTavern，AIO Hub 深度兼容其格式，包括世界书、角色卡、快捷动作和正则脚本。

### 🧰 30+ 专业工具集

从 API 测试、Git 分析、正则应用到视频转写、颜色提取、JSON 格式化——一应俱全。后面有详细分类。

### 🪟 自由窗口管理

受够了固定的布局？随便拖。聊天区域、工具栏、输入框——**任何组件都可以拖拽成独立窗口**，贴边自动吸附，重启后还记得位置。

### 🧩 插件系统（三种类型）

- **JS 插件**：写一段前端脚本，动态注册功能，即写即用。
- **Native 插件**：Rust 编写的高性能后端插件。
- **Sidecar 插件**：用 Python、Go 等任何语言编写独立进程，系统统一管理。

### 🎨 极致视觉定制

- 原生 Windows Mica / macOS Vibrancy 毛玻璃特效
- 动态视频/图片轮播壁纸，自适应取色
- 内置 CSS 编辑器，想改哪改哪（搞个赛博朋克主题？安排）
- 明/暗主题一键切换

### 🌍 跨平台支持

| 平台    | 安装格式                              |
| :------ | :------------------------------------ |
| Windows | `.exe` 安装包                         |
| macOS   | `.dmg`（支持 Apple Silicon 和 Intel） |
| Linux   | `.AppImage` / `.flatpak`              |

### 📱 移动端（进行中）

Android / iOS 版本正在开发中，与桌面端共享核心逻辑，触摸屏交互深度重构。

## 架构一览

> 这一节是**面向你**的通俗版解释，不是技术文档。

AIO Hub 由三部分组成：

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  你看到的界面  │────▶│ 背后的核心逻辑  │────▶│  系统底层能力  │
│  (Vue 3 前端)  │     │  (TypeScript)  │     │  (Rust 后端)  │
└──────────────┘     └──────────────┘     └──────────────┘
```

1. **你看到的界面** — 基于 Vue 3，所有按钮、输入框、聊天气泡、设置面板都在这里。
2. **背后的核心逻辑** — TypeScript 编写的业务逻辑，负责处理 LLM 请求、工具之间的通信、状态管理等。
3. **系统底层能力** — Rust 编写的原生能力，负责高性能计算（如正则批量处理、Git 分析）、文件系统操作、系统 API 调用（如 OCR）等。这部分你看不到，但它是"快"的秘诀。

这三层通过 Tauri 桌面框架粘合在一起，最终打包成一个可安装的桌面应用。

> 对于开发者：如果你对插件开发或自定义工具感兴趣，参考 `docs/guide/` 下的开发指南。

## 使用场景

### 场景一：AI 角色扮演与小说创作

> 你想和 AI 角色进行沉浸式对话，同时管理庞大的人物设定。

1. 先配置一个 LLM 服务（比如 OpenAI 或 DeepSeek）。
2. 在"智能体编辑器"中创建一个角色，写好性格设定。
3. 在"ST 世界书编辑器"中创建世界书，为重要关键词绑定背景故事。
4. 开始对话。当谈到某个地点或人物时，世界书会自动注入设定。
5. 使用对话分支管理查看对话的树状历史，随时回退到旧分支重来。
6. 最后导出 PNG 角色卡分享给朋友。

### 场景二：日常开发辅助

> 你正在开发一个项目，需要频繁地查日志、比代码、转格式。

1. 把 AIO Hub 放在副屏，随时 AI 对话解决技术问题。
2. 需要比对两个配置文件？拖拽进去，差异一目了然。
3. 批量处理日志中的正则匹配？用"正则应用器"建一条流水线。
4. 想了解项目的 Git 历史？交给"Git 分析器"秒出可视化结果。
5. 媒体文件太大需要压缩？FFmpeg 助手一键处理。

### 场景三：资料整理与内容处理

> 你有一堆图片、PDF、录音需要整理。

1. "智能 OCR"扫描图片中的文字，多引擎识别，准确率高。
2. "音视频转写"把会议录音转成文字稿。
3. "内容去重"清理重复段落。
4. "数据筛选"从大量 JSON 中提取你需要的内容。
5. 把整理好的资料丢到"资产管理器"中统一管理。

### 场景四：多窗口高效工作流

> 你需要同时盯着 AI 回答、查 API 文档、看日志。

1. 把主聊天窗口的"对话区域"拖拽成独立悬浮窗，放到屏幕左侧。
2. 打开"API 测试器"，也拖拽出来放到屏幕右侧。
3. 主窗口切到"LLM 请求检查器"监控中间流量（如开启了本地代理）。
4. 三个窗口随时可见，无需切换 Tab。

### 场景五：自定义插件扩展

> 你需要一个内置应用不支持的功能。

1. 打开文档，学习 JS 插件开发（只需要一点前端经验）。
2. 编写一个简单的 JS 文件，通过 `activate` 钩子注册你的自定义功能。
3. 插件系统中开启它，即刻生效。
4. 如果是复杂功能，用 Rust 写 Native 插件，或者用 Python 写 Sidecar 插件。

## 常见问题

### Q1：AIO Hub 需要联网吗？

**部分功能可以离线使用。** 系统提供的本地能力（正则处理、代码格式化、OCR 本地引擎、文件操作等）完全离线可用。但 LLM 对话需要联网访问 API 服务。

### Q2：我的 API Key 安全吗？

**安全的。** 所有 API Key 和配置数据都存储在本地，不会上传到任何服务器。你可以随时在设置中查看和管理 Key。

### Q3：AIO Hub 和 SillyTavern 什么关系？

AIO Hub **不是** SillyTavern 的替代品，两者定位不同。但 AIO Hub 深度兼容 SillyTavern 的格式生态，包括世界书（`.json`）、角色卡（PNG V2/V3）、快捷动作、正则脚本等。如果你是从 SillyTavern 转过来了，大部分素材可以直接导入。

### Q4：能在手机上用吗？

**桌面版已稳定可用，移动端正在开发中。** Android 和 iOS 版本已进入开发阶段，与桌面端共享核心逻辑，UI 针对触摸屏重新设计。

### Q5：如何获取更新？

**直接下载最新版本安装即可，配置和数据不会丢失。** 关注 GitHub Releases 页面即可获取版本通知。推荐下载 **Pre-release 预览版**，更新更频繁，功能最新。
