# 外观与壁纸

> 外观设置让 AIO Hub 的视觉风格完全由你掌控——从明暗主题到毛玻璃透明度，从动态壁纸到原生窗口特效。本篇将逐一介绍每个配置项的效果和用法。

---

## 入口

1. 点击标题栏右侧的 ⚙️ 图标进入设置。
2. 在设置左侧导航栏中，点击「主题壁纸外观」选项卡。

该面板采用**双栏布局**：左栏管理壁纸，右栏控制界面质感和窗口特效。

```
┌──────────────────────┬────────────────────────────┐
│   壁纸管理（左栏）   │   界面质感（右栏上半）     │
│                      │   窗口特效（右栏下半）     │
├──────────────────────┴────────────────────────────┤
│   💡 小提示：所有调整都即时预览，无需保存。       │
└───────────────────────────────────────────────────┘
```

## 主题设置

主题设置位于「通用设置」面板中，是调整视觉风格的第一步。

| 模式                | 说明                                     |
| :------------------ | :--------------------------------------- |
| **浅色 (Light)**    | 经典的明亮界面，适合白天或光线充足的环境 |
| **深色 (Dark)**     | 沉浸式的暗色界面，适合夜晚或保护视力     |
| **跟随系统 (Auto)** | 根据操作系统的外观设置自动切换亮暗主题   |

## 主题色设置

在「主题色配置」面板中，你可以定制应用的关键色彩。

| 颜色类型   | 预设数量 | 说明                                     |
| :--------- | :------- | :--------------------------------------- |
| **主题色** | 17 种    | 含 Element 标准色 + VOCALOID/UTAU 风格色 |
| **成功色** | 11 种    | 成功状态提示颜色                         |
| **警告色** | 14 种    | 警告状态提示颜色                         |
| **危险色** | 14 种    | 危险/错误状态提示颜色                    |
| **信息色** | 6 种     | 普通信息提示颜色                         |

每种颜色都支持：

- 从预设色板中选择
- 使用原生拾色器自定义 HEX 颜色
- 一键恢复默认值

> 当「主题色提取」开启时，主题色会自动被壁纸提取色覆盖，并显示「壁纸提取中」标签。

## 壁纸管理

壁纸管理位于「主题壁纸外观」面板的左栏。

### 壁纸开关

面板顶部的全局开关控制壁纸功能的启用/停用。关闭后壁纸和特效均不显示。

### 壁纸模式

| 模式         | 说明                                |
| :----------- | :---------------------------------- |
| **静态壁纸** | 固定显示一张图片作为壁纸            |
| **轮播模式** | 定时切换多张壁纸，支持随机/顺序播放 |

**轮播模式**下，你可以：

- 设置轮播间隔（1-1440 分钟）
- 暂停/继续播放
- 随机播放或顺序播放，切换时重新洗牌
- 预览缩略图列表，点击即可切换到指定壁纸
- 通过控制栏按钮（上一张、暂停/继续、下一张）实时控制

### 壁纸来源

| 来源         | 说明                                                   |
| :----------- | :----------------------------------------------------- |
| **内置壁纸** | 从预设的壁纸网格中选择，点击即应用                     |
| **自定义**   | 静态模式下选择单个图片文件；轮播模式下选择一个图片目录 |

### 填充模式

| 模式               | 效果                             | 适用场景               |
| :----------------- | :------------------------------- | :--------------------- |
| **覆盖 (Cover)**   | 等比例缩放填满全屏，可能裁剪边缘 | 最常用，适合大多数图片 |
| **包含 (Contain)** | 完整显示整张图片，留黑边         | 需要看到完整图片内容   |
| **拉伸 (Fill)**    | 拉伸填满，可能变形               | 适合无缝纹理           |
| **平铺 (Tile)**    | 重复拼接所有图片                 | 适合小尺寸图案或纹理   |

**拼贴模式**提供额外参数调整：

- **缩放**：0.1x - 3x，控制平铺单元的放大/缩小
- **旋转**：0° - 360°，每个单元的旋转角度
- **水平翻转 / 垂直翻转**

### 清除壁纸

点击「清除壁纸」按钮可以移除当前设置的壁纸，恢复无壁纸状态。

> 壁纸设置界面左上角有**预览区域**，实时显示当前壁纸的效果。点击预览区域可以**查看大图**。

## 界面质感

界面质感部分通过「UI 特效」全局开关控制。关闭后，所有透明度与模糊效果都被禁用，界面回到纯色风格。

> 💡 **禁用 UI 特效可以略微提高性能**，特别是低配置设备或集显机型上。

### UI 基础不透明度

滑块控制（0 - 1，步长 0.05），影响所有 UI 元素的整体透明度。值越小，壁纸透出的越多。

### 分离窗口不透明度（悬浮化的组件）

滑块控制分离出去的悬浮窗口的内部元素透明度（默认 0.95），与主窗口独立设置。

> ⚠️ **重要提示**：在主窗口中，由于底层有壁纸和窗口背景色叠加，UI 不透明度可以调得非常低（如 0.1）依然能看清。但**分离的悬浮组件窗口**底层通常是完全透明的，如果此值设置过低，窗口内的文本和图标将因失去背景托底而变得难以辨认。建议保持在 0.8 - 1.0 之间。

### 背景色叠加

背景色叠加是毛玻璃效果的关键补充。开启后，在壁纸之上叠加一层颜色，通过混合模式产生丰富的视觉效果。

| 配置项                 | 说明                                                                     |
| :--------------------- | :----------------------------------------------------------------------- |
| **叠加颜色**           | 选择颜色，或使用**屏幕吸色（EyeDropper API）**取色，或从当前壁纸自动提取 |
| **自动从壁纸提取颜色** | 开启后，切换壁纸时自动提取主色调用于叠加                                 |
| **混合模式**           | 支持 16 种 CSS 混合模式，用于定义背景色与壁纸的融合方式（详见下表）      |
| **叠加不透明度**       | 控制叠加层的透明程度                                                     |

#### 支持的混合模式

| 模式            | 说明     | 模式           | 说明   |
| :-------------- | :------- | :------------- | :----- |
| **normal**      | 正常     | **soft-light** | 柔光   |
| **multiply**    | 正片叠底 | **difference** | 差值   |
| **screen**      | 滤色     | **exclusion**  | 排除   |
| **overlay**     | 叠加     | **hue**        | 色相   |
| **darken**      | 变暗     | **saturation** | 饱和度 |
| **lighten**     | 变亮     | **color**      | 颜色   |
| **color-dodge** | 颜色减淡 | **luminosity** | 亮度   |
| **color-burn**  | 颜色加深 | **hard-light** | 强光   |

### 主题色提取

此功能独立于背景色叠加。开启后，当切换壁纸时，系统会自动提取壁纸中的亮眼颜色，**应用到全局主题色**（按钮、选中态、链接等）。

四种提取策略：

| 策略                         | 效果                                   |
| :--------------------------- | :------------------------------------- |
| **柔和 (Muted)** — 推荐      | 提取最温和、不刺眼的颜色，适合日常使用 |
| **鲜艳 (Vibrant)**           | 提取最鲜艳的颜色，更具活力             |
| **亮部鲜艳 (Light Vibrant)** | 从图片亮部区域提取鲜艳色               |
| **暗部鲜艳 (Dark Vibrant)**  | 从图片暗部区域提取鲜艳色               |

> 可以随时在 **主题色设置** 面板中看到当前提取的颜色预览。

### 壁纸不透明度

独立控制壁纸层的透明度（0 - 1）。降低此值可以让壁纸更淡，使背景色叠加层或纯色背景更明显。

### UI 模糊（毛玻璃）

全局开关 + 强度滑块（0-50px）。开启后，所有 UI 卡片、面板、对话框都会应用毛玻璃模糊效果，透出壁纸。

### 边线

- **边线不透明度**（0 - 1）：控制 UI 元素边框线的透明度
- **边线宽度**（0 - 5px）：控制边框线的粗细

> 将边线不透明度设为 0 可隐藏所有边框，获得更纯的毛玻璃效果。

### 代码块背景不透明度

控制代码编辑器（如 CSS 覆盖编辑器、LLM 返回的代码块）的背景透明度，使其更好地融入整体风格而不是一块"死沉"的实色。

## 窗口特效（实验性）

::: warning 注意
窗口特效依赖于操作系统的原生能力，目前仅在 **Windows 10+** 和 **macOS** 上可用。Linux 系统因兼容性问题会**自动禁用**并显示提示。
:::

| 特效                 | 说明                                           |
| :------------------- | :--------------------------------------------- |
| **无**               | 不启用任何窗口原生特效                         |
| **模糊 (Blur)**      | Windows 窗口背景模糊                           |
| **亚克力 (Acrylic)** | Windows 10+ 亚克力效果，更丰富的模糊与纹理混合 |
| **云母 (Mica)**      | Windows 11 云母材质效果，性能更好的透明背景    |

### 窗口背景不透明度

独立开关 + 滑块控制。需要开启窗口特效且该开关启用才生效。降低此值以透出桌面或窗口后的内容。

### 窗口阴影

控制窗口是否显示阴影效果。关闭后可获得更简洁的视觉风格。

## 常见问题

### Q：调整设置后如何恢复默认？

每个滑块和开关都可以手动调回默认值。主题色部分提供「重置为默认」按钮，壁纸可以点击「清除壁纸」。

### Q：为什么我的窗口特效选项是灰色的？

如果你是 Linux 用户，窗口特效因兼容性问题被自动禁用。Windows 用户请确认操作系统版本 ≥ Windows 10。

### Q：毛玻璃效果导致性能下降怎么办？

关闭「UI 特效」全局开关即可禁用所有透明度与模糊效果，界面回到纯色风格，性能开销降到最低。

### Q：如何让应用看起来完全透明？

可以尝试以下组合：

1. 开启壁纸
2. 开启「UI 特效」
3. 将「UI 基础不透明度」调到较低值
4. 开启「UI 模糊」
5. 在窗口特效中选择「亚克力」或「云母」
6. 开启窗口背景不透明度并调低
