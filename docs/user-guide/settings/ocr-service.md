# 云端 OCR 服务

> 如果你需要使用智能 OCR 工具的云端识别引擎，需要先在这里配置 OCR 服务渠道。

---

## 入口

1. 点击标题栏右侧的 ⚙️ 图标进入设置。
2. 在左侧导航栏中选择 **云端 OCR 服务**。

---

## 服务配置

### 标准服务商

| 字段                 | 说明                                |
| :------------------- | :---------------------------------- |
| **服务名称**         | 自定义名称                          |
| **服务类型**         | 选择预设的 OCR 服务商（如百度云等） |
| **API 端点**         | 服务地址                            |
| **API Key / Secret** | 认证凭据                            |

### 自定义 API 支持

如果标准服务商不满足需求，你可以配置**自定义 API**：

<div v-pre>
| 配置项             | 说明                            |
| :----------------- | :------------------------------ |
| **自定义 API URL** | 请求目标地址                    |
| **HTTP 方法**      | GET / POST 等                   |
| **请求头**         | 自定义请求头                    |
| **请求体模板**     | 支持 `{{imageBase64}}` 变量占位 |
| **结果提取路径**   | 从响应 JSON 中提取识别结果      |
</div>
---

## 性能与限制

| 配置项       | 说明                                     |
| :----------- | :--------------------------------------- |
| **并发数**   | 1-10，控制图片块并行处理数               |
| **请求延迟** | 0-5000ms，每次请求间的延时，避免触发限流 |

> 所有配置变更会在 1 秒防抖后**自动保存**。

---

## 常见问题

### Q：OCR 服务配置用于哪里？

主要用于**智能 OCR** 工具的云端识别引擎。如果你只使用本地 OCR，则无需配置此项。
