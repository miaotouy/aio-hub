# 云端 OCR 服务

云端 OCR 服务用于给「智能 OCR」工具提供远程识别渠道。你可以使用预设服务商，也可以配置兼容自己接口的自定义 OCR API。

## 入口

1. 点击标题栏右侧的设置图标进入设置。
2. 在左侧导航栏中选择「云端 OCR 服务」。
3. 点击新增按钮，从预设模板或空白配置开始。

配置完成后，回到 [智能 OCR](../tools/smart-ocr/index.md) 工具，在引擎设置中选择云端 OCR。

## 服务类型

| 类型            | 状态 | 说明                         |
| :-------------- | :--- | :--------------------------- |
| 百度智能云 OCR  | 可用 | 需要 API Key 和 API Secret   |
| 腾讯云 OCR      | 预留 | 当前云端 Runner 中暂未实现   |
| 阿里云 OCR      | 预留 | 当前云端 Runner 中暂未实现   |
| 自定义 OCR 服务 | 可用 | 通过请求模板适配你自己的接口 |

如果你使用腾讯云或阿里云，当前更推荐通过「自定义 OCR 服务」接入你自己的代理接口，或使用 VLM 方案完成识别。

## 从预设创建

1. 点击新增按钮。
2. 选择一个预设服务商。
3. 系统会自动填入服务类型、默认端点和图标。
4. 填写服务名称和凭据。
5. 调整并发数和请求延迟。
6. 表单会在变更后自动保存。

百度云配置需要同时填写 API Key 和 API Secret。系统会用它们获取访问令牌并发起 OCR 请求。

## 自定义 OCR API

选择「自定义 OCR 服务」后，需要配置完整请求模板。

### URL 地址

填写接口地址。支持变量占位符，例如：

```text
https://{{host}}/api/ocr
```

变量需要在变量配置中添加，发送请求时会替换为当前值。

### HTTP 方法

当前支持 `POST` 和 `PUT`。大多数 OCR 接口使用 `POST`。

### 请求头

可以添加任意 Header。常见示例：

```text
Content-Type: application/json
Authorization: Bearer {{apiKey}}
```

如果接口使用固定 Key，可以把它放在变量中，避免每次改模板。

### 请求体模板

请求体必须是有效 JSON。使用 `{{imageBase64}}` 表示当前图片块的 base64 数据。

示例：

```json
{
  "image": "{{imageBase64}}",
  "detect_direction": true
}
```

发送时系统会直接替换变量，不会通过 `fetch(dataUrl)` 读取图片，因此不会触发 data URL 的 CSP 问题。

### 结果路径

结果路径用于从返回 JSON 中提取识别文本。例如：

| 返回结构                            | 结果路径        |
| :---------------------------------- | :-------------- |
| `{ "data": { "text": "..." } }`     | `data.text`     |
| `{ "result": [{ "text": "..." }] }` | `result.0.text` |
| `{ "words": "..." }`                | `words`         |

如果路径填错，智能 OCR 会收到空结果或提取失败。

## 并发与限流

智能 OCR 会把图片切成多个块。云端 OCR 服务中的并发和延迟会影响这些块的请求节奏。

- 并发数：同时处理多少个图片块，范围 1-10。
- 请求延迟：每次请求之间等待多少毫秒，范围 0-5000ms。

建议：

- 免费或限流严格的接口：并发 1-2，延迟 500-1500ms。
- 稳定内网服务：并发 3-6，延迟 0-200ms。
- 识别失败率升高时：先降低并发，再增加延迟。

## 启用与排序

左侧服务列表支持启用/禁用和拖拽排序。禁用的服务不会作为可选渠道使用；排序可以帮助你把常用服务放在前面。

配置会自动保存。删除服务前会弹出确认框，删除后不可恢复。

## 常见问题

### 为什么云端 OCR 没有出现在智能 OCR 中？

确认该服务已启用，并且智能 OCR 的引擎类型选择了云端 OCR。如果仍不可见，重新打开智能 OCR 工具触发配置加载。

### 百度云提示凭据缺失怎么办？

百度云需要 API Key 和 API Secret 两项。只填其中一个会请求失败。

### 自定义接口一直提取不到文本？

先用 API 测试工具请求同一个接口，确认返回 JSON 结构。然后把结果路径改成能准确定位文本的字段，例如 `data.text` 或 `result.0.text`。

### 并发越高越好吗？

不一定。过高并发容易触发服务商限流，也会让失败重试变多。优先选择稳定的并发，再追求速度。

### 腾讯云和阿里云为什么显示但不可用？

界面保留了服务商类型和预设，但当前云端 OCR Runner 中对应调用尚未实现。可以使用自定义 OCR 服务接入自己的代理接口。
