# 工具管理

工具管理用于控制主页工具卡片的显示状态和排列顺序。它不会卸载工具，也不会删除工具数据，只影响你在主页和常用入口中看到工具的方式。

## 入口

1. 点击标题栏右侧的设置图标进入设置。
2. 在左侧导航栏中选择「工具模块」。
3. 右侧会显示全部已注册工具卡片。

页面顶部会显示已启用数量和工具总数，例如 `18/34`。

## 显示或隐藏工具

每个工具以卡片形式展示。卡片被选中时，会在主页显示；取消选中后，会从主页工具网格中隐藏。

常用操作：

- 点击单个工具卡片：切换显示或隐藏。
- 点击「全选」：把所有工具显示到主页。
- 点击「全不选」：隐藏所有工具卡片。
- 查看顶部数量：确认当前启用数量。

隐藏工具适合把低频工具收起来，让主页更清爽。工具本身仍然存在，重新勾选即可恢复。

## 调整工具顺序

工具卡片左侧有排序手柄。将鼠标移到卡片上，拖动排序手柄即可调整顺序。

排序会影响：

- 主页工具卡片排列。
- 已打开工具侧边栏中的默认顺序。
- 部分工具选择器中使用的工具顺序。

拖拽结束后会自动保存。如果顺序没有变化，系统不会重复写入配置。

## 重置顺序

点击「重置顺序」会清除自定义排序，恢复为项目默认工具顺序。

它只重置顺序，不会改变工具显示或隐藏状态。也就是说，如果某个工具已经被你隐藏，重置顺序后它仍然保持隐藏。

## 注意事项

- 工具管理不会关闭正在运行的后台服务。
- 隐藏工具不等于禁用工具启动项；自动启动行为请到「启动项管理」中单独调整。
- 插件工具或新增工具注册后，可能会以默认显示状态出现在列表中。
- 如果工具列表顺序异常，可以先点「重置顺序」，再重新拖拽常用工具到前面。

## 常见问题

### 隐藏工具后还能通过链接打开吗？

通常可以。隐藏只影响主页展示，不代表工具路由或内部能力被移除。

### 为什么隐藏工具后它仍在后台运行？

该工具可能注册了启动项，或被其他模块调用。请同时检查「启动项管理」和相关工具设置。

### 工具排序没有保存怎么办？

确认拖拽的是排序手柄，并在拖拽结束后看到成功提示。如果仍未保存，查看日志配置中是否有设置写入错误。

### 可以恢复默认显示状态吗？

目前页面提供顺序重置。显示状态可以通过「全选」快速恢复为全部显示，再按需要手动隐藏少数工具。
