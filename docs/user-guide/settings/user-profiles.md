# 用户档案管理

> 用户档案用于 LLM 聊天场景，为 AI 对话提供用户侧的描述信息，包括头像、显示名称及个性化行为配置。

---

## 入口

1. 点击标题栏右侧的 ⚙️ 图标进入设置。
2. 在左侧导航栏中选择 **用户档案管理**。

---

## 功能说明

### 档案列表

侧边栏按最近使用排序，显示：

- 头像
- 名称
- 描述（截断）
- 创建日期

### 编辑档案

点击档案进入编辑模式，支持：

- **头像上传/清除**
- **显示名称与简介**
- **富文本样式行为配置**：定义用户在对话中的发言样式
- **正则预置管理**：管理用户消息的预处理规则

### 常用操作

| 操作         | 说明                                 |
| :----------- | :----------------------------------- |
| **新建档案** | 弹窗输入名称、显示名、简介、头像 URL |
| **删除档案** | 确认后删除到回收站                   |
| **打开目录** | 直接打开档案配置文件所在目录         |

> 所有表单变更会在 1 秒防抖后自动保存。
