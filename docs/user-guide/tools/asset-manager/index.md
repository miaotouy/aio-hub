# 📦 资产管理器 - 快速上手

## 概述

资产管理器是应用的**中央资产仓库**，所有工具（如 LLM 聊天、智能 OCR、媒体生成器等）导入的文件都会统一存储在这里。你可以把它理解为应用的"资源管理器"——浏览、搜索、预览和管理所有已导入的图片、视频、音频和文档。

> **核心特性：统一存储，跨工具复用。** 你在聊天中上传的图片、OCR 扫描的文档、智能生成的媒体文件，都会被集中管理，随时可查，无需重复导入。

::: tip 核心场景

- 想回顾之前聊天中上传的某张图片
- 批量清理不再需要的导入文件
- 查看每个资源的来源信息，追溯"这个文件是哪来的"
- 发现重复文件并释放磁盘空间
  :::

### 资产类型

资产管理器支持以下五种资产类型：

| 类型     | 说明                          | 预览方式                           |
| -------- | ----------------------------- | ---------------------------------- |
| **图片** | PNG / JPG / WebP / GIF 等     | 内置图片查看器（缩放、旋转、翻页） |
| **视频** | MP4 / WebM / AVI 等           | 内置视频播放器（倍速、画中画）     |
| **音频** | MP3 / FLAC / WAV / OGG 等     | 内置音频播放器（播放列表）         |
| **文档** | PDF / TXT / Markdown / 代码等 | 文档预览器（源码/渲染切换）        |
| **其他** | 未分类的文件格式              | 暂不支持预览                       |

## 导入资产

资产管理器本身不提供独立的上传入口——资产是通过各个工具的使用过程中**自动导入**的。

### 方式一：拖放文件到应用窗口

任意支持文件拖放的工具（如 LLM 聊天、媒体生成器等），将文件从系统文件管理器拖拽到应用窗口，文件会自动导入资产库。

### 方式二：粘贴剪贴板内容

复制图片或文件后，切换到 LLM 聊天等工具，使用 `Ctrl + V`（或 `Cmd + V`）粘贴，文件会自动导入。

### 方式三：工具自动生成

部分工具会自动生成资产并存入资产库：

- **LLM 聊天** — 聊天中上传或生成的图片
- **智能 OCR** — 扫描识别后保存的文档
- **媒体生成器** — AI 生成的图片和视频

#### 资产统计 (侧边栏)

侧边栏顶部实时显示当前资产库的：

- **总数量**：所有已导入文件的总个数。
- **总大小**：所有资产占用的总磁盘空间。

### 切换视图模式

工具栏右侧提供了两种视图模式：

- **网格视图** — 卡片式展示，适合浏览图片等视觉类资产
- **列表视图** — 表格形式展示，包含文件名、类型、大小、来源、导入时间等详细信息

网格视图下还可以调整卡片尺寸：**大卡片** / **中卡片** / **小卡片**。

### 分组浏览

通过工具栏的"分组"下拉菜单，可以按不同维度组织资产：

| 分组方式       | 说明             | 示例                               |
| -------------- | ---------------- | ---------------------------------- |
| **按月份**     | 按导入月份分组   | 2025年03月、2025年02月...          |
| **按类型**     | 按文件类型分组   | 图片、视频、音频、文档...          |
| **按来源**     | 按导入方式分组   | 本地文件、剪贴板、网络、AI 生成... |
| **按来源模块** | 按来源工具分组   | LLM 聊天、智能 OCR...              |
| **不分组**     | 平铺展示所有资产 | —                                  |

### 排序方式

工具栏支持多种排序维度，帮助你快速定位文件：

- **按时间** (默认)：按导入时间倒序排列，最新的文件在前。
- **按名称**：按文件名的字母顺序排列。
- **按大小**：按文件体积大小排列。

### 搜索与筛选

- **关键词搜索** — 在工具栏的搜索框中输入文件名关键词，实时过滤
- **侧边栏筛选** — 可按文件类型、来源方式、来源模块进行精确筛选
- **组合筛选** — 搜索关键词与多种筛选条件可叠加使用

## 基本操作

### 查看和预览资产

- **图片**：单击预览，在图片查看器中缩放、旋转，支持同组图片翻页浏览
- **视频**：单击打开视频播放器，支持倍速播放、画中画、截图
- **音频**：单击开始播放，支持播放列表顺序切换
- **文档**：单击在弹窗中预览，支持源码/渲染模式切换

### 多选操作

资产卡片/行支持多选：

- **单击**：选中/取消单个资产
- **`Ctrl + 单击`（或 `Cmd + 单击`）**：切换单个资产的选中状态
- **`Shift + 单击`**：范围选中连续资产

选中资产后，工具栏会显示"删除选中"和"取消选择"按钮。

### 文件操作 (高级功能)

工具栏左侧的 **文件操作** 下拉菜单提供了一些维护资产库的进阶工具：

- **查找重复文件**：扫描全库，识别内容完全相同（哈希一致）的文件。发现重复后，界面上会高亮标记重复资产。
- **自动选中副本**：在发现重复文件后，此功能会自动选中那些“多余”的副本（保留每组中最新的一份），方便你一键清理。
- **清除重复标记**：取消界面上的重复高亮显示。
- **重建索引**：如果发现资产列表显示异常或搜索不准确，可以使用此功能重新扫描元数据并重建查询索引。

### 操作菜单

每个资产卡片或行都提供操作菜单（点击右上角的 `···` 按钮）：

- **预览** — 打开对应的内置查看器（图片、视频、音频或文档）
- **打开所在目录** — 在系统文件管理器中定位原始文件
- **删除** — 将该资产从库中移除并移动到系统回收站

### 删除资产

支持单个删除和批量删除：

1. 选中要删除的资产（可多选）
2. 点击工具栏的 **删除选中** 按钮
3. 确认删除操作
4. 文件会被移至系统回收站（可恢复）

::: warning
删除资产会同时删除对应的原始文件。在确认删除前，请确保文件不再需要。
:::

### 打开文件所在目录

右键资产 → **打开所在目录**，系统会打开文件资源管理器并定位到该资产的实际存储位置，方便将文件复制到其他地方使用。
