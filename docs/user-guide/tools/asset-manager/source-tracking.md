# 📦 资产管理器 - 来源追踪

## 概述

资产管理器的每个资产都带有"身份证明"——它记录了资产从哪来（来源方式）、由哪个工具导入（来源模块）。来源追踪功能让你一目了然地追溯每个文件的"出生证明"。

> **核心价值：面对大量资产时，快速回答"这个文件是哪来的"。**

## 查看资产来源

### 在列表视图中查看

切换到**列表视图**（工具栏右侧的列表按钮），表格中的"来源"列会清晰显示每个资产的来源信息：

```
┌──────────────┬────────────┬──────────────────┐
│   文件名      │   类型     │      来源         │
├──────────────┼────────────┼──────────────────┤
│ 设计稿.png   │   图片     │ LLM 聊天 · 本地   │
│ 报告截图.jpg  │   图片     │ 智能 OCR · 本地   │
│ 脑暴图.png   │   图片     │ 剪贴板            │
│ 语音记录.mp3  │   音频     │ LLM 聊天 · 生成   │
└──────────────┴────────────┴──────────────────┘
```

来源信息的显示格式为：**来源模块 · 来源方式**，例如 `LLM 聊天 · 本地` 表示该资产是通过 LLM 聊天工具从本地文件导入的。

### 在网格视图中查看

网格视图卡片右上角有一个 `···` 按钮，点击后在下拉菜单中可以看到资产的详细来源信息。

## 来源信息详解

每个资产的来源由两部分组成：

### 来源方式 (Origin Type)

表示资产是通过什么途径进入系统的：

| 来源方式     | 标签        | 说明                                 |
| ------------ | ----------- | ------------------------------------ |
| **本地**     | `HardDrive` | 从本地文件系统导入（拖放或浏览选择） |
| **剪贴板**   | `Clipboard` | 通过粘贴复制内容导入                 |
| **网络**     | `Globe`     | 从网络 URL 下载获取                  |
| **生成内容** | `Sparkles`  | 由 AI 或其他工具自动生成             |

### 来源模块 (Source Module)

表示是哪个工具导入了该资产，例如 `LLM 聊天`、`智能 OCR`、`媒体生成器` 等。系统会自动识别来源模块的名称并显示为友好的中文标签。

## 追溯原始文件

### 打开文件所在目录

如果需要定位资产的原始存储路径：

1. 右键资产 → 选择 **打开所在目录**
2. 系统会打开文件资源管理器，定位到该资产的实际存储位置

这个操作对于需要手动复制文件到其他地方使用的场景非常有用。

### 通过文件类型定位

在侧边栏的"来源方式"筛选区域，可以按导入途径筛选资产：

- **全部** — 显示所有来源方式的资产
- **本地** — 仅显示从文件系统导入的资产
- **剪贴板** — 仅显示粘贴导入的资产
- **网络** — 仅显示从网络获取的资产
- **生成内容** — 仅显示由 AI 生成的资产

## 来源信息管理

### 按来源模块筛选

侧边栏第三个筛选区域会列出所有有资产的来源模块及其数量统计。你可以点击任意模块，只看该工具导入的资产。

例如，点击"💬 LLM 聊天"只显示聊天中上传的图片和文件，快速定位聊天记录中的附件。

### 按来源方式分组

在工具栏的"分组"下拉菜单中选择**按来源**，资产会按照导入途径分组展示：

- **本地文件** — 从文件系统导入的资产
- **剪贴板** — 通过粘贴导入的资产
- **网络** — 从网络获取的资产
- **未知来源** — 来源信息不明确的资产

选择**按来源模块**，资产则会按照导入的工具分组。

## 典型场景

### 场景一：找到聊天中发送的某张图

1. 在侧边栏"来源模块"区域点击"💬 LLM 聊天"
2. 资产列表只显示聊天中导入的资产
3. 输入文件名关键词进一步搜索
4. 找到后可以直接预览或打开所在目录

### 场景二：清理剪贴板导入的临时文件

1. 在侧边栏"来源方式"区域点击"剪贴板"
2. 筛选出所有通过粘贴导入的资产
3. 选择不需要的文件，批量删除

### 场景三：了解哪些工具最"吃"存储

1. 切换分组方式为"按来源模块"
2. 查看各模块组的资产数量和文件大小
3. 快速了解哪个工具产生了最多的资产
