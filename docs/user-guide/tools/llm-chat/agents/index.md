# 🤖 智能体 (Agent) 概览

智能体 (Agent) 是 LLM Chat 的核心概念——它是**角色设定 + 模型参数 + 工具权限 + 预设消息**的完整配置包。你可以在不同 Agent 之间快速切换，以适应不同的对话场景。

## 什么是 Agent？

一个典型的 Agent 包含以下核心组件：

| 组件 | 说明 | 详细指南 |
| :--- | :--- | :--- |
| **🆔 基本信息** | 名称、图标、描述、分类标签 | [基础设定](./editor-guide#1-基础设定-basic) |
| **👤 角色设定** | 系统提示词、角色人格定义、预设消息 | [参数配置](./parameters) |
| **⚙️ 模型参数** | 关联的服务渠道 (Profile)、模型 ID、Temperature 等 | [参数配置](./parameters) |
| **🔧 工具配置** | 工具调用开关、自动批准规则、并行执行控制 | [工具调用](../tool-calling/index) |
| **📖 关联资源** | 世界书、快捷操作、用户档案绑定 | [关联资源](./parameters#关联资源) |
| **🧩 扩展配置** | 插件扩展启用状态、环境增强 | [环境增强](./editor-guide#6-环境增强-environment) |
| **🎨 输出表现** | 渲染样式、正则替换、视觉化指南 | [输出与显示](./editor-guide#7-输出与显示-output) |

## 切换 Agent

系统提供了多种方式在不同智能体之间无缝切换：

- **左侧面板**: 点击 Agent 列表中的条目即可立即切换当前会话的 Agent。
- **快速切换菜单**: 在消息输入框上方点击 🤖 图标，呼出快速切换下拉菜单。
- **自动切换策略**: 当你切换到某个会话（Session）时，系统会自动切换到该会话最后一次使用的 Agent。

## 数据隔离与安全性

为了保证不同角色扮演或任务场景的独立性，每个 Agent 拥有独立的数据存储空间：

1. **配置隔离**: 所有的提示词、参数和变量均独立存储。
2. **资产隔离**: 每个 Agent 拥有独立的 [资产目录](./assets)，避免不同角色间的媒体资源混淆。
3. **安全删除**: 当你删除一个 Agent 时，其配置文件和关联资产目录会被移入系统回收站，而非直接永久抹除，方便误删恢复。
