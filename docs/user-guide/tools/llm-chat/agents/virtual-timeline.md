# 虚拟时间线 (Virtual Timeline)

虚拟时间线是为角色扮演（Roleplay）场景设计的核心功能，允许你为 Agent 设定一个独立于现实世界的“游戏时间”。

## 1. 核心概念

通过虚拟时间线，你可以让 Agent 意识到当前处于虚拟世界的哪个时刻，从而在对话中表现出符合时间背景的行为（如：深夜时的困倦、特定节日的庆祝）。

- **虚拟基准点**: 虚拟世界的起始时间（如：2077年10月23日 09:00）。
- **现实基准点**: 对应现实世界的时刻。系统支持一键同步为“当前现实时间”。
- **时间流速倍率**: 虚拟时间相对于现实时间的流逝速度。
  - `1.0`: 虚拟时间与现实同步。
  - `2.0`: 现实过去 1 小时，虚拟世界过去 2 小时。
  - `0.0`: 虚拟时间静止。

## 2. 宏系统联动

虚拟时间线最强大的地方在于它与[宏系统](../macro-system/index)的深度集成。你可以直接在系统提示词或预设消息中使用以下宏：

- `{{time}}`: 虚拟世界的当前时间（24小时制）。
- `{{datetime_cn}}`: 虚拟世界的中文完整日期时间。
- `{{shichen}}`: 虚拟时刻对应的中国传统时辰（如：子时、寅时）。
- `{{virtual_day_progress}}`: 当前虚拟天内的进度百分比。

## 3. 实时预览

在 Agent 编辑器的虚拟时间线面板中，提供了一个**实时预览区域**：

- 你可以输入任何包含时间宏的文本。
- 系统会每秒自动刷新预览结果，让你直观看到在当前流速下，宏是如何展开的。

## 4. 使用场景示例

### 场景 A：异世界冒险

- **设定**: 虚拟基准点设为“1202年”，流速设为 `3.0`（快节奏冒险）。
- **提示词**: `当前时间是 {{datetime_cn}}，你正处于 {{shichen}}。`
- **效果**: 随着现实时间的推移，Agent 会自动感知到虚拟世界天色的变化，并在回复中提及“夜深了”或“正午阳光刺眼”。

### 场景 B：历史模拟

- **设定**: 虚拟基准点设为“1945年”，流速设为 `0.0`。
- **效果**: 无论现实过去多久，Agent 始终认为自己处于那个特定的历史时刻。
