# ❓ 常见问题

## 连接与配置

### Q: 为什么发送消息后一直没回复？

A: 常见原因和解决方法：

1. **检查网络连接**：确保你的网络可以访问 AI 服务提供商的 API
2. **检查 API Key**：进入「设置 → AI 服务配置」确认 API Key 有效且未过期
3. **检查模型选择**：确认已为 Agent 选择了正确的模型
4. **查看错误信息**：如果消息内容区域显示红色错误提示（如"请求超时"、"API Key 无效"等），根据提示排查
5. **超时设置**：在聊天设置 → 请求设置中增加超时时间

### Q: 如何配置代理？

A: 当你的网络需要通过代理访问 AI 服务时：

1. 进入「设置 → AI 服务配置」
2. 在对应的服务渠道中配置代理地址（如 `127.0.0.1:7897`）
3. 部分服务支持独立的代理设置

## 费用与 Token

### Q: 使用 LLM Chat 需要付费吗？

A: 应用本身是免费的。但调用 AI 模型的 API 需要你自己承担费用——你需要准备各 AI 服务商的 API Key 并充值。

### Q: 怎么控制 Token 消耗？

A: 以下方法可以有效减少 Token 使用：

1. **缩短 Max Tokens**：在 Agent 参数中设置较小的输出上限
2. **启用上下文压缩**：长对话后使用压缩节点将历史摘要化
3. **关闭不需要的处理器**：如不需要世界书或知识库，在管道配置中关闭
4. **定期创建新会话**：避免单次对话积累过多历史

## 功能使用

### Q: 如何导入 SillyTavern 角色卡？

A: 支持导入 PNG 格式的 SillyTavern V2 角色卡：

1. 在创建 Agent 时选择「导入 SillyTavern 角色卡」
2. 选择 `.png` 文件
3. 系统会自动解析角色信息并创建 Agent

### Q: 分支太多怎么管理？

A: 切换到**对话树图视图**（点击中栏顶部的视图切换按钮），以力导向布局查看所有分支。你还可以：

- 使用消息导航器前进/后退浏览
- 在树图中拖拽节点调整布局
- 删除不需要的分支（点击节点上的删除按钮）

### Q: 为什么我的消息没有发送出去？

A: 可能的原因：

1. **当前会话正在生成中**：等待当前生成完成后再发送
2. **快捷键冲突**：确认快捷键设置（`Ctrl+Enter` 或 `Enter`）
3. **输入框为空**：不能发送空白消息

### Q: 如何撤销误操作？

A: 在聊天窗口中使用 `Ctrl + Z` 撤销最近一次操作，如删除消息、编辑内容等。撤销历史基于当前会话维护。

## 数据与隐私

### Q: 我的对话数据存储在哪里？

A: 所有对话数据均存储在**本地**：

- 应用数据目录下的 `AppData` 文件夹
- 会话以分离的索引和详情文件存储
- Agent 和用户档案有独立的存储目录

数据不会自动上传到云端。

### Q: 如何备份我的数据？

A: 推荐方法：

1. **导出 Agent**：在 Agent 编辑器中导出为 JSON 或 ZIP
2. **导出会话**：将会话导出为 Markdown 或 JSON
3. **手动备份**：复制 `AppData/llm-chat/` 目录下的所有文件

### Q: 导出数据会包含 API Key 吗？

A: **不会**。Agent 导出文件仅包含配置信息（模型 ID、预设消息等），不会包含你的 API Key 或其他敏感凭据信息。

## 性能问题

### Q: 对话超过 100 条消息后变得很卡怎么办？

A: 优化建议：

1. **创建新会话**：将当前会话作为备份，新建会话继续对话
2. **使用上下文压缩**：将早期对话压缩为摘要
3. **调整虚拟列表渲染**：在设置中减小 `virtualListOverscan` 值
4. **启用 HTML 预览冻结**：自动冻结非最新消息的 HTML 预览

### Q: 生成速度太慢怎么办？

A: 可能的优化方向：

1. **选择更快的模型**：如 GPT-4o-mini、Claude Haiku 等
2. **缩短 Max Tokens**：减少单次生成的输出长度
3. **关闭流式平滑化**：在渲染设置中关闭，可减少渲染开销
4. **检查网络延迟**：使用更靠近服务端的代理节点

## 兼容性

### Q: 支持哪些 AI 服务商？

A: 支持所有兼容 OpenAI API 格式的服务商，包括：

- OpenAI (GPT-4o, GPT-4o-mini 等)
- Anthropic (Claude 系列)
- Google (Gemini 系列)
- SiliconFlow、DeepSeek、Moonshot 等国内服务商
- 以及任何兼容 OpenAI 代理格式的自部署服务

### Q: 支持流式输出吗？

A: 默认开启流式输出。你可以在聊天设置 → UI 偏好中关闭它（不推荐，会降低响应速度感知）。
