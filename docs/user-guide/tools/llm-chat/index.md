# 🤖 LLM 对话 — 快速上手

::: tip 功能概要
LLM 对话是 AIO Hub 中最核心、功能最丰富的工具。它不仅仅是聊天窗口，更是一个支持**树状分支对话**、**智能体 (Agent) 系统**、**世界书 (Worldbook)**、**上下文管道**等高级功能的综合对话引擎。
:::

## 核心特性一览

| 特性          | 说明                                                   |
| ------------- | ------------------------------------------------------ |
| 🧬 树状分支   | 每条消息都可以产生多个回复分支，自由切换和对比         |
| 🤖 智能体系统 | 预设角色、系统提示词、模型参数，一键切换不同 Agent     |
| 📖 世界书     | 类似角色扮演游戏的背景设定库，自动匹配关键词注入上下文 |
| 🔧 上下文管道 | 可编排的上下文处理器链，灵活控制发送给 LLM 的内容      |
| 📎 附件管理   | 支持图片、文件、音频等附件，无缝集成资产管理器         |
| 📤 导入导出   | 支持 JSON / Markdown 导出，跨设备迁移数据              |

## 开始第一次对话

1. **打开 LLM 对话工具**：从 **🏠 主页** 的工具网格中点击 **💬 LLM 聊天** 卡片。打开后，它会出现在左侧侧边栏的标签列表中。
2. **确认已配置 AI 服务**：如果尚未配置，请先点击窗口右上角的 **⚙️ 设置** 图标，前往「AI 服务配置」添加 API Key。
3. **选择或创建 Agent**：首次使用会自动创建默认的「助手」Agent，你也可以从智能体面板新建
4. **在输入框打字**：底部的消息输入框支持多行文本，按 `Ctrl + Enter` 发送（可在设置中改为 `Enter`）

## 界面布局速览

LLM 对话采用经典的三栏布局：

- **左栏**：功能切换区。通过顶部标签页在 **🔮 智能体列表** 和 **⚙️ 模型参数** 之间切换。
- **中栏**：核心对话区。包括消息展示区域（线性列表或树状图）和底部的消息输入框。
- **右栏**：会话管理区。列出所有历史会话，支持搜索、重命名和删除。

顶部标题栏支持**拖拽分离**，将对话区域独立成浮动窗口。

## 快速技巧

- **切换视图**：点击中栏顶部的按钮可在「线性消息列表」和「对话树图」间切换
- **上下文分析器**：点击工具栏的 🔍 按钮查看即将发送给 LLM 的完整上下文
- **快捷键**：`Ctrl + Enter` 发送、`Ctrl + Z` 撤销、`Ctrl + Shift + Z` 重做
