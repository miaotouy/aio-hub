# 💬 消息操作概览

LLM Chat 的消息系统基于**树状分支结构**——每条消息都可以产生多个回复，形成一棵对话树。这让你可以自由探索不同的对话方向，而不会丢失已有的内容。

## 核心交互

### 发送消息

- **快捷键**: 默认使用 `Ctrl + Enter` 发送，`Enter` 换行。
- **分支逻辑**: 如果你当前正在查看的历史节点已经有了后续回复，发送新消息会自动开启一个**新的平行分支**。

### 输入工具栏

输入框上方集成了多种增强工具：

- 📎 **附件管理**: 添加图片、文档或音频。
- 🤖 **快速切换**: 实时更换当前使用的 Agent。
- ⚡ **快捷动作**: 触发预设的模板指令。
- 🔍 **上下文预览**: 在发送前检查即将提交给 LLM 的完整 Prompt。

## 消息树与分支

分支是本系统的灵魂。你可以从任意历史节点“嫁接”出新的对话路径：

- **分支选择器**: 消息卡片上下方的标签页，用于在平行分支间快速跳转。
- **导航控制**: 使用消息导航器 (Navigator) 在父子节点间前进或后退。
- **全局视野**: 切换到 [对话树图](../sessions/tree-graph) 视图，以可视化方式管理复杂的对话拓扑。

## 撤销与重做 (Snapshot)

系统为每个会话维护了全量快照历史：

- 使用 `Ctrl + Z` 撤销上一次的消息编辑、删除或分支切换。
- 撤销操作不仅恢复文本，还会恢复整个消息树的物理结构。
