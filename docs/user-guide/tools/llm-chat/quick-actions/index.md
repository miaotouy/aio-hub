# 快捷操作 (Quick Actions) 概览

快捷操作是一套强大的文本模板与指令系统，允许用户在输入框中通过点击预设按钮，快速对文本进行包装、转换或直接发送复杂指令。

## 核心概念

-   **多级关联机制**：快捷操作不是孤立存在的，它们可以关联到不同的维度：
    -   **全局 (Global)**：在所有会话中均可见。
    -   **智能体 (Agent)**：仅在特定智能体激活时显示。
    -   **用户档案 (User Profile)**：随用户身份切换而变化。
-   **三层覆盖策略**：当多个维度的快捷操作组同时存在时，系统会按优先级聚合显示，确保最相关的操作触手可及。

## 主要功能

1.  **文本包装**：利用模板语法，一键为选中内容添加 HTML 标签、Markdown 引用或特定的指令前缀。
2.  **指令发送**：配置“自动发送”后，点击按钮即可立即向模型发起请求，无需手动敲击回车。
3.  **多组管理**：支持按功能领域（如代码助手、翻译工具、创意写作）对操作进行分组管理。

## 交互位置

在聊天界面的输入框工具栏中，你会看到“闪电”图标。点击即可弹出当前可用的快捷操作列表。
