# 管理与导入导出

快捷操作管理器提供了一个直观的界面，用于组织和维护你的操作库。

## 管理器界面

你可以通过侧边栏或设置菜单进入“快捷操作管理”：
-   **左侧列表**：管理快捷操作组（Set）。支持新建、重命名、克隆和批量删除。
-   **中间列表**：管理组内的具体操作条目。
-   **右侧编辑器**：编辑选定操作的名称、模板内容、自动发送开关及后处理规则。

## 分组与排序

-   **组启用状态**：你可以暂时禁用某个组，使其不在输入框的快速列表中显示。
-   **排序**：系统会根据组的创建或更新时间进行默认排序。建议将最常用的操作放在独立的组中并保持启用。

## 导入与导出

为了方便分享和备份，快捷操作支持标准的 JSON 格式：
-   **单组导出**：导出选中的单个快捷操作组。
-   **批量导出**：在“批量管理”模式下，勾选多个组一键导出为压缩包或合并的 JSON。
-   **导入**：
    -   支持点击“导入”按钮选择文件。
    -   支持将 JSON 文件直接拖入管理器界面的“投放区”实现快速导入。

## 默认预设

系统内置了两组常用的快捷操作：
1.  **代码助手**：包含隐藏代码块包装、完整代码更新请求等。
2.  **通用工具**：包含 HTML 折叠标签包装、Markdown 引用包装等。
