# 模板语法与宏联动

快捷操作的核心在于其灵活的模板语法，允许你定义复杂的文本结构。

## {{input}} 占位符

`{{input}}` 是快捷操作中最核心的占位符：
-   **行为**：
    -   如果输入框中有选中的文本，`{{input}}` 将被替换为选中的文本。
    -   如果输入框中没有选中的文本，`{{input}}` 将被替换为输入框中的全部文本。
-   **用途**：实现“包装”模式。例如模板内容为 `请翻译以下内容：\n\n{{input}}`。

## 文本后处理 (Line Processing)

系统支持对生成的每一行内容进行自动化处理：
-   **行前缀/后缀**：例如为每一行添加 `> ` 实现批量引用。
-   **正则替换**：支持在包装完成后，对结果进行二次正则清洗。

## 与宏系统联动

快捷操作完整集成了系统的**宏引擎 (Macro Engine)**：
-   **动态注入**：你可以在模板中使用任何系统宏，如 `{{time}}`、`{{date}}`、`{{user}}` 等。
-   **变量操作**：通过 `{{getvar::key}}` 获取会话变量，或通过 `{{setvar::key::value}}` 在执行快捷操作的同时修改会话状态。
-   **自动补全**：在快捷操作编辑器中，输入 `{{` 即可触发宏的自动补全提示。

## 包装模式 vs 指令模式

-   **包装模式**：通常包含 `{{input}}`，用于处理用户现有的输入。
-   **指令模式**：不包含 `{{input}}`，直接发送预定义的指令（如“继续生成”、“总结全文”）。配合“自动发送”开关，可实现一键触达。
