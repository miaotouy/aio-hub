# 对话树图 (Tree Graph)

对话树图是 LLM Chat 的核心特性之一，它将线性的对话历史转变为可视化的拓扑结构。通过树图，您可以直观地查看、管理和操作复杂的对话分支。

## 1. 视图切换

在对话区域的工具栏中，您可以随时在 **线性列表视图** 和 **树图视图** 之间切换：

- **线性列表**：适合阅读长篇对话，侧重于当前分支的上下文连贯性。
- **树图视图**：适合管理多分支探索，侧重于对话的结构化组织和非线性跳转。

## 2. 布局模式

树图支持三种不同的布局算法，以适应不同的使用场景：

| 模式                   | 说明                                         | 适用场景                       |
| :--------------------- | :------------------------------------------- | :----------------------------- |
| **动态树状 (Tree)**    | 确定性的层级排列，结合物理碰撞防止节点重叠。 | 结构清晰的对话，层级关系明确。 |
| **物理引力 (Physics)** | 基于 D3-force 的力导向布局，带有模拟重力。   | 探索性对话，节点分布更自然。   |
| **静态树状 (Static)**  | 纯几何计算布局，不包含物理模拟。             | 节点极多时的性能优化模式。     |

> [!TIP]
> 您可以在树图右侧的控制栏中点击布局图标（网格/分享/减号）循环切换模式，或在“视图设置”中指定默认布局。

## 3. 节点操作

树图中的每个节点代表一条消息，您可以直接对其进行交互：

### 基础交互

- **单击选择**：选中节点并在右侧或悬浮窗查看详情。
- **双击跳转**：将当前对话分支切换到该节点所在路径。
- **右键菜单**：提供“设为当前分支”、“禁用/启用节点”、“剪掉分支”等快捷操作。

### 拖拽与移动

- **单点拖拽**：直接拖动节点可手动调整其位置。
- **子树拖拽**：按住修饰键（默认 `Shift`，可在设置中修改）拖动节点，将连带其所有子孙节点一起移动。
- **重置布局**：如果布局变得混乱，点击控制栏的“刷新”图标可一键重置到算法初始位置。

### 分支嫁接 (Grafting)

这是树图最高级的操作之一。您可以改变对话的历史逻辑：

1. 从一个节点的底部连接点按住并拖出连线。
2. 将连线指向另一个节点。
3. **单点移动**：仅将该节点移动到新父节点下。
4. **子树嫁接**：按住修饰键（默认 `Shift`）进行连接，将整个子树整体迁移到新分支。

## 4. 历史面板 (History Panel)

树图的操作是**非破坏性**的，所有结构变更都会记录在会话历史中。

- **撤销/重做**：支持对节点编辑、移动、嫁接、删除等操作进行无限次撤销（快捷键 `Ctrl+Z` / `Ctrl+Y`）。
- **查看历史**：点击控制栏的“时钟”图标打开历史面板。
- **操作类型**：
  - `BRANCH_GRAFT`：分支嫁接
  - `NODE_MOVE`：节点移动
  - `NODE_EDIT`：内容编辑
  - `NODE_TOGGLE_ENABLED`：启用/禁用切换
- **状态快照**：历史面板会显示每次操作影响的节点数，并允许您一键跳转回任意历史状态。

## 5. 辅助功能

- **HUD 信息面板**：显示当前 FPS、节点总数、布局模式和缩放比例。
- **小地图 (MiniMap)**：在节点过多时辅助导航。
- **自动定位**：点击控制栏的“准星”图标，视口将自动平滑滚动并对齐到当前激活的叶子节点。
- **调试模式**：为高级用户提供，显示 D3 物理引擎的力向量、速度和节点状态数据。
