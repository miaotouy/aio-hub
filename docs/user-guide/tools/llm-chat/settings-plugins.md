# 🔌 插件与扩展设置

LLM Chat 支持通过多种扩展机制增强对话能力，包括**上下文管道处理器**、**正则替换规则**、**快捷操作**以及**插件设置注册**。

## 上下文管道 (Context Pipeline)

上下文管道是一系列按顺序执行的**处理器 (Processor)**，共同负责将原始消息组装成最终发送给 LLM 的请求上下文。

### 默认处理器

| 处理器                      | 作用                                                 |
| --------------------------- | ---------------------------------------------------- |
| **Session Loader**          | 从树状会话中提取当前活跃路径的线性消息列表           |
| **Regex Processor**         | 执行 Global/Agent/User 三层正则替换规则              |
| **Injection Assembler**     | 处理复杂的预设消息注入（含深度注入和锚点定位）       |
| **Knowledge Processor**     | 执行 RAG 检索，将知识库条目注入上下文                |
| **Transcription Processor** | 处理附件的转写结果及纯文本文件的内容提取             |
| **Worldbook Processor**     | 扫描关键词并注入关联的世界书条目                     |
| **Token Limiter**           | 核心预算管理，按优先级裁剪超长上下文                 |
| **Async Task Processor**    | 汇总并注入关联的异步任务执行状态                     |
| **Variable Processor**      | 处理会话变量的动态展开与状态更新                     |
| **Message Formatter**       | 调整消息结构以符合特定厂商的 API 要求                |
| **Asset Resolver**          | 最后一步：将资产占位符转换为最终的 Base64 或资源 URL |

### 启用/禁用处理器

在「聊天设置 → 上下文管道」中可以：

1. **开关处理器**：每个处理器都可以独立启用或禁用
2. **调整顺序**：拖拽处理器条目改变执行顺序
3. **恢复默认**：一键重置为出厂配置

## 正则替换规则 (Regex Rules)

正则替换规则允许你在消息发送前自动对文本进行清洗和转换。

### 使用场景

- **格式清理**：自动移除冗余的空行、特殊字符
- **角色扮演增强**：自动替换人称代词、语气词
- **内容过滤**：屏蔽敏感词或不符合要求的表达
- **格式统一**：标准化标点符号、排版

### 配置方式

1. 打开聊天设置 → 文本处理
2. 找到「全局文本替换」折叠区域
3. 点击展开，添加正则规则：
   - **名称**：规则的标识
   - **匹配模式**：正则表达式
   - **替换文本**：替换后的内容（支持捕获组引用）
   - **启用状态**：可临时禁用某条规则

### 绑定模式

- **跟随消息配置**：每条消息使用其生成时的 Agent/User 自身的正则配置
- **使用当前会话配置**：所有消息统一使用当前会话的 Agent/User 配置

## 快捷操作 (Quick Actions)

快捷操作让你将常用的指令、模板或宏保存为一键发送的快捷按钮。

### 配置入口

在聊天设置 → 快捷操作中管理你的快捷操作库：

1. **创建操作组**：将多个相关快捷操作组织在一起
2. **添加操作**：定义操作的名称、内容方法
3. **全局关联**：将操作组关联到全局（所有会话可用）或特定 Agent

### 使用快捷操作

在输入框上方的工具栏中点击 ⚡ 按钮，选择要发送的快捷操作。

## 插件设置注册

第三方插件可以通过 `registerSettingsSection` 和 `registerSettingItem` API 将自己的设置分组注入到聊天设置对话框中。这允许插件扩展无缝集成到 LLM Chat 的设置界面中。

### 注册方式

```typescript
import { registerSettingsSection } from "@/tools/llm-chat/composables/settings/usePluginSettings";

registerSettingsSection({
  title: "我的插件",
  icon: MyIcon,
  items: [
    {
      id: "myPluginOption",
      label: "插件选项",
      component: "ElSwitch",
      modelPath: "plugins.myPlugin",
    },
  ],
});
```

注册后，设置项会自动出现在聊天设置的对应分组中。
