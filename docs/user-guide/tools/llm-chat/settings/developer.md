# <LIcon name="TerminalSquare" size="28" /> 开发者选项

开发者选项为高级用户和插件开发者提供了一些深度的调试工具和 UI 强制覆盖功能。

## 高级控制

- **启用调试模式 (Debug Mode)**:
  - **视觉反馈**: 开启后，界面将透出更多的底层信息。例如：在高级关系图中显示节点 ID、在消息操作栏显示“查看原始 JSON”按钮。
  - **用途**: 当你遇到生成异常、渲染错误或需要向开发者提交 Bug 报告时，开启此模式可以获取更详尽的诊断数据。
- **强制允许原生缩放 (Native Resize)**:
  - **功能**: 控制分离模式（悬浮窗）下的组件窗口是否响应操作系统的原生缩放行为。
  - **场景**: 开启后，窗口的隐形边框将响应鼠标悬停时的缩放指示并允许拖拽缩放。这主要用于调试透明无边框窗口的边界响应。在正常使用中，由于组件通常带有自定义的缩放控件，该选项一般无需开启，以保持界面的视觉纯净。

---

::: danger 警告
**安全提示**: 开启调试模式可能会在日志或 UI 中暴露 API Key 的部分信息或敏感的上下文数据。在公共场合演示或分享截图前，请务必确认已关闭此选项。
:::
