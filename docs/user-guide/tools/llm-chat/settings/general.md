# <LIcon name="Bot" size="28" /> 通用设置

通用设置模块包含了 LLM Chat 的全局基础配置，主要用于定义系统的默认行为和兜底逻辑。

## 核心配置

### 全局默认模型 (Default Model)

- **说明**: 指定在新建智能体（Agent）或开启新会话（Session）时，系统预设的首选模型。
- **兜底机制**: 当一个智能体未指定特定模型，或者某个会话的专属模型失效时，系统将自动切换到此模型以确保对话不中断。
- **配置建议**:
  - 建议选择一个**全能型且响应稳定**的模型，如 Gemini 3 flash 这种多模态模型，能很好的支持各种需求。
  - 如果你主要在本地运行，也可以将其设为你的主力本地模型。

---

::: tip 提示
**优先级说明**: 全局默认模型的优先级最低。系统始终优先使用 **[智能体配置](../agents/editor-guide)** 或 **会话临时参数** 中指定的模型。
:::
