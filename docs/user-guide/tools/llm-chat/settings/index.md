# ⚙️ 聊天设置概览

LLM Chat 提供了极其丰富的设置项，涵盖了从底层模型参数、上下文管道编排到顶层 UI 交互的各个方面。

## 设置入口

点击聊天界面右上角的 **⚙️ (设置)** 按钮即可打开全局聊天设置对话框。设置项按功能划分为以下核心分区：

| 分区 | 说明 | 详细指南 |
| :--- | :--- | :--- |
| **通用 (General)** | 快捷键、发送逻辑、自动生成标题、请求超时等基础配置。 | [UI 偏好与通用设置](./ui-preferences) |
| **显示与渲染** | 消息字体、流式输出、Markdown 渲染与视觉化指南开关。 | [渲染与显示](./rendering) |
| **上下文管道** | 启用/禁用处理器、调整执行顺序、Token 预算管理。 | [上下文管道](../context-pipeline/index) |
| **文本处理** | 管理全局正则替换规则。 | [全局正则规则](./regex-global) |
| **转写与语音** | 配置 Whisper 语音转写及附件处理策略。 | [转写与语音配置](./transcription) |
| **插件与扩展** | 第三方插件注入的自定义设置项。 | [插件设置](./plugins) |

## 配置层级

系统采用 **“就近原则”** 的配置覆盖策略：
1. **全局设置**: 在聊天设置对话框中配置，作为所有会话的基准。
2. **Agent 设置**: 在 [Agent 编辑器](../agents/editor-guide) 中配置，可覆盖全局的模型、正则和管道设置。
3. **会话设置**: 某些特定参数（如临时切换的模型）仅对当前会话生效。
