# <LIcon name="Zap" size="28" /> 快捷操作

快捷操作（Quick Actions）是提升聊天效率的神器。通过预设指令组，你可以将复杂的 Prompt 封装为工具栏上的一个按钮，实现一键总结、润色、续写等操作。

## 关联与应用

- **全局关联快捷操作**:
  - **说明**: 在此下拉框中选择一个或多个已创建的快捷操作组。
  - **生效范围**: 选中的组将作为“全局工具”出现在 **所有会话** 的输入框上方工具栏中。
  - **建议**: 关联那些通用的指令组（如：通用润色、代码优化）。

## 快捷操作库管理

这是一个完整的管理终端，支持以下高级功能：

- **创建与编辑**: 自由创建操作组，并为每个组定义多个按钮（Actions）。
- **拖拽排序**: 调整操作组在列表中的顺序，决定其在工具栏的显示位置。
- **导入与导出**:
  - 支持导出为 `.json` 文件进行备份或分享。
  - 支持从本地文件或剪贴板导入他人分享的指令集。
- **批量管理**: 进入批量模式可快速删除或导出多个操作组。
