# <LIcon name="Keyboard" size="28" /> 快捷键

通过自定义快捷键和操作修饰键，你可以获得更符合直觉的交互体验，尤其是在处理复杂的长对话和关系图时。

## 消息输入习惯

- **发送消息策略**:
  - **Ctrl/Cmd + Enter (推荐)**: 适合习惯在输入框中进行多行编辑和排版的用户。单按 `Enter` 仅执行换行。
  - **Enter**: 适合即时通讯风格的快速对话。换行需使用 `Shift + Enter`。

## 高级关系图修饰键 (Graph Modifiers)

这些高级设置专门为 **[会话树状图 (AOE)](../sessions/tree-graph)** 视图设计，用于提升节点操作的效率：

- **历史记录控制**:
  - **撤销/重做**: 自定义在关系图中回溯操作的按键组合（如 `Ctrl + Z` / `Ctrl + Y`）。
- **子树联动操作**:
  - **拖拽子树修饰键**:
    - **功能**: 当你拖动一个父节点时，若按住此修饰键（如 `Alt`），其下的所有子节点将保持相对位置同步移动。
    - **用途**: 快速整理复杂的分支布局。
  - **嫁接子树修饰键**:
    - **功能**: 在重新连线（改变父节点）时，若按住此键，整个子树将作为一个整体迁移到新父节点下。
    - **用途**: 快速调整对话分支的逻辑归属。

---

::: info 提示
快捷键设置在修改后立即生效，你可以在关系图视图中实时测试不同修饰键的手感。
:::
