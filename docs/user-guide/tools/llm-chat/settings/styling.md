# <LIcon name="Palette" size="28" /> 样式设置

样式设置模块赋予了你完全掌控聊天界面视觉表现的能力。通过编写自定义 CSS，你可以改变消息渲染的每一个细节。

## 全局消息样式 (Global Markdown Style)

提供了一个集成语法高亮的 CSS 编辑器，用于定义消息的基础样式：

- **层级逻辑**: 此处的 CSS 会作为全局样式表注入。它遵循 **“Agent 优先”** 原则——如果你在某个智能体的设置中定义了私有样式，私有样式将覆盖此处的全局设置。
- **编辑器功能**:
  - 支持实时预览。
  - 内置了对常用消息元素的样式模板。
- **典型应用**:
  - 修改特定语言代码块的背景色。
  - 自定义 `blockquote`（引用）的边框和字体颜色。
  - 调整 Markdown 列表的缩进和行高。
  - 为特定的 HTML 标签（如 `<kbd>`）添加装饰。

---

::: tip 提示
**善用变量**: 系统暴露了大量的 CSS 变量（如 `--card-bg`, `--primary-color`），使用这些变量编写样式可以确保你的自定义外观能自动适配“明亮/暗黑”模式及“界面质感”调整。

详情参考：**[CSS 变量与宏指南](../macro-system/css-variables-guide)**。
:::
