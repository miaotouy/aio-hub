# 转写与语音配置 (Transcription)

LLM Chat 集成了强大的语音转写功能，允许你直接向 Agent 发送语音消息，或对上传的音视频附件进行内容提取。

## 1. 核心转写配置

- **转写引擎**: 
  - **Whisper (Local)**: 使用本地硬件加速进行转写，保护隐私且无需联网。
  - **OpenAI Whisper (API)**: 使用云端 API，转写速度快且准确率高。
- **语言检测**: 
  - `Auto`: 自动检测说话人的语言。
  - `zh/en/ja...`: 强制指定特定语言以提高准确率。
- **提示词 (Prompt)**: 为转写引擎提供上下文提示（如：专有名词、特定术语），以减少转写错误。

## 2. 附件处理策略

在设置中你可以定义如何处理不同类型的附件：

- **自动转写**: 开启后，拖入输入框的音频/视频文件会自动开始后台转写。
- **文本提取**: 对于 PDF、Word、TXT 等文档，系统会自动提取纯文本内容。
- **注入模式**: 
  - **占位符模式**: 在文本中保留 `[附件:文件名]` 标记。
  - **全量注入**: 将转写或提取的所有文本直接追加到上下文末尾。

## 3. 语音输入交互

- **录音模式**: 
  - **点击录音**: 点击开始，再次点击结束并转写。
  - **长按录音**: 类似移动端体验，松开即发送。
- **静音自动断句**: 开启后，当检测到长时间沉默时，系统会自动停止录音并开始转写。
