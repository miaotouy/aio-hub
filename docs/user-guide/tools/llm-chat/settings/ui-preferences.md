# UI 偏好与通用设置

控制聊天界面的基础交互逻辑和视觉表现。

## 1. 通用设置 (General)

- **发送快捷键**: 
  - `Ctrl + Enter`: 默认发送方式，`Enter` 换行。
  - `Enter`: 交换后 `Enter` 直接发送，`Ctrl + Enter` 换行。
- **自动生成标题**: 开启后，新会话的前几轮对话后，系统会调用 LLM 自动总结并更新会话标题。
- **请求超时**: 设置 API 请求的全局超时时间（单位：毫秒）。
- **重试次数**: 当请求失败（如网络抖动）时，系统自动重试的次数。

## 2. 列表与交互

- **虚拟渲染开关**: 控制长对话的消息列表是否启用虚拟渲染技术。建议始终开启以保证性能。
- **自动滚动**: 开启后，当新消息生成时，列表会自动滚动到底部。
- **打字机效果 (Stream)**: 开启后，LLM 的回复将以流式逐字显示。
- **消息时间戳**: 控制是否在每条消息下方显示发送/生成的具体时间。

## 3. 性能优化

- **增量保存间隔**: 设置会话数据自动持久化到本地数据库的时间间隔。
- **图片压缩**: 开启后，上传或粘贴的大图在发送给 LLM 之前会进行自动压缩，以节省 Token 并加快上传速度。
