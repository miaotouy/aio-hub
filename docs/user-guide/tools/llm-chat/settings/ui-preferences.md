# <LIcon name="Settings2" size="28" /> 界面偏好

界面偏好模块专注于提升你的阅读舒适度，并允许你深度定制聊天窗口的交互细节。

## 视觉可见性 (Visibility)

- **统计信息**:
  - **会话 Token 统计**: 实时监控当前对话的消耗。
  - **块级 Token 统计**: 极其精细地显示每一个代码块或工具调用消耗的 Token，帮助你理解模型开销。
- **元数据**:
  - 可选显示消息的时间戳和具体使用的模型名称，方便复盘。

## 排版与阅读 (Typography)

- **文本微调**: 除了基础的 **字号** 和 **行高**，你还可以调整 **字间距**。这对于解决某些中文字体显得过于拥挤的问题非常有效。
- **宽度限制**:
  - **功能**: 在大显示器上，开启“限制内容宽度”并配合“居中显示”，可以防止视线横向跨度过大，显著缓解阅读疲劳。

## 滚动与吸附 (Scrolling)

- **触底吸附策略**:
  - **功能**: 当你向上翻阅历史记录时，系统会自动暂停自动滚动。
  - **阈值**: 你可以自定义“吸附距离”。只要你滚动回距离底部 N 像素以内，系统就会重新激活自动跟随，确保你能无缝衔接最新的生成内容。

## 高级交互与质感 (Advanced UI)

- **毛玻璃效果**:
  - **头部透明度/模糊**: 聊天区域的顶部标题栏支持独立调节。你可以将其设为完全透明并配合高强度模糊，营造出通透的视觉质感。
- **悬浮窗控制**:
  - **分离手柄**: 开启后，输入框和消息区会出现 **<LIcon name="ExternalLink" size="14" /> (分离)** 图标，允许你将特定区域拖拽为独立的桌面悬浮窗。
  - **壁纸透出**: 在分离模式下，你可以选择是否让窗口背景呈现出应用全局的壁纸。

---

::: tip 提示
界面偏好的大部分修改都是即时生效的，你可以一边聊天一边调整，直到找到最舒服的视觉组合。
:::
