# <LIcon name="BookMarked" size="28" /> 世界书

世界书（Worldbook / Lorebook）是管理长期记忆和复杂设定的核心组件。它通过关键词触发机制，在对话过程中动态注入相关的背景知识。

## 匹配策略

- **全局关联世界书**:
  - **说明**: 在此选择的世界书将作为“常驻背景”应用于 **所有会话**。
  - **场景**: 适用于存放通用的世界观（如：魔法系统规则）、常驻角色信息或全局避雷指南。
- **禁用递归扫描**:
  - **说明**: 开启后，被触发并注入上下文的条目内容本身不会再触发其他条目。
  - **建议**: 除非你有复杂的条目嵌套需求，否则建议开启以节省 Token 并防止逻辑死循环。
- **默认扫描深度**:
  - 控制扫描历史消息的范围。数值越大，模型越能记住很久以前提到的关键词，但计算开销也越大。

## 世界书库管理

提供了一个可视化的库管理终端：

- **快速导入**:
  - 支持直接拖拽 `.json` 文件导入。
  - **兼容性**: 深度适配 **SillyTavern (酒馆)** 的 World Info 格式。
- **总览与管理**:
  - 一键查看所有已导入世界书的条目数量、触发权重等元数据。
  - 点击 **“管理”** 进入全屏编辑器，可对单个条目进行精细化的正则匹配、概率触发及插入位置配置。
