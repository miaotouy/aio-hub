# ⌨️ 快捷键与技巧

## 常用快捷键

### 消息操作

| 快捷键             | 功能                               |
| ------------------ | ---------------------------------- |
| `Ctrl + Enter`     | 发送消息（默认）                   |
| `Enter`            | 换行（发送设为 `Ctrl + Enter` 时） |
| `Ctrl + Z`         | 撤销操作                           |
| `Ctrl + Shift + Z` | 重做操作                           |
| `Esc`              | 取消编辑、关闭弹窗                 |

### 快捷键设置

你可以在聊天设置 → 快捷键中切换：

- **发送键**：可选 `Ctrl + Enter` 或 `Enter`
- **换行键**：自动与发送键互补

## 视图相关快捷键

### 对话树图 (FlowTreeGraph)

在高级关系图视图中，以下修饰键控制节点操作：

| 操作     | 默认修饰键 | 说明                          |
| -------- | ---------- | ----------------------------- |
| 拖拽子树 | `Alt`      | 按住 Alt 拖拽节点移动整个子树 |
| 嫁接子树 | `Alt`      | 按住 Alt 连线可嫁接整个子树   |

## 隐藏功能与效率技巧

### 上下文分析器

点击工具栏的 🔍 按钮打开上下文分析器。这里你可以看到：

- **结构化视图**：完整的消息列表，标注每条消息的来源（预设/历史/世界书等）
- **原始请求视图**：发送给 LLM 的原始 JSON 请求
- **宏调试视图**：宏展开前后的对比
- **分析图表**：各模块的 Token 占比

这是调试对话质量和排查问题的利器。

### 快速 Agent 切换

在输入框上方使用 🤖 下拉菜单，不离开对话界面即可切换 Agent。

### 消息导航器

左侧的 `MessageNavigator` 组件提供：

- 消息历史的前进/后退浏览
- 当前节点的所有分支列表
- 快速跳转到任意消息

### 对话树图模式

点击视图切换按钮，从「线性列表」切换到「树图视图」可以看到消息的完整分支结构。这对理解复杂的多分支对话非常有帮助。

### 模板技巧

#### 使用宏动态化提示词

```markdown
当前时间是 {{time}}，日期是 {{date_cn}}。
你正在与 {{user}} 对话。
用户的背景：{{persona}}
```

#### 随机化内容

```markdown
{{random::请用正式语气回复::请用轻松的语气回复::请用诗意的语言回复}}
```

#### 条件注入（结合预设消息的模型匹配）

为不同模型准备不同的预设消息版本，让同一条系统提示词在不同模型下展现不同效果。

### 快速导出

右键点击会话选择「导出会话」可快速导出当前活动路径的 Markdown。右键点击消息选择「导出分支到此」可以只导出到该消息为止的对话。

### 暂停生成

在 AI 回复生成过程中，可以随时点击停止按钮中止生成。已生成的内容不会被丢弃。

### 性能优化

- **CSS 原生虚拟渲染**：通过 `content-visibility: auto` 利用浏览器原生能力跳过屏外元素渲染，相比传统 JS 虚拟滚动更平滑。
- **HTML 预览冻结**：自动冻结非最新消息的 HTML 预览以节省内存。
- **渲染节流**：控制流式渲染的频率，配合 `requestAnimationFrame` 降低 CPU 负载。
- **消息数量**：注意不要在单个会话中积累过多消息，适时创建新会话。
