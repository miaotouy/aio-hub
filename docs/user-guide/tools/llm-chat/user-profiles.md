# 👤 用户档案 (User Profile)

用户档案定义了你在对话中扮演的角色身份。当你与 AI 对话时，用户档案的内容会通过宏 `{{persona}}` 注入到上下文中，让 AI 知道正在与谁交谈。

## 为什么需要用户档案？

在不同的对话场景中，你可能希望 AI 以不同的方式理解你的身份：

- **角色扮演**：你扮演一位调查员、骑士或学生
- **专业对话**：你是一位开发者、设计师或作家
- **多用户使用**：不同家庭成员使用各自的档案

用户档案让这一切变得简单——无需修改 Agent 的提示词，只需要切换档案即可。

## 创建档案

1. 在聊天设置 → 或通过 Agent 编辑器中的档案管理入口
2. 点击「新建档案」
3. 填写名称和内容
4. 可选：设置头像图标

### 档案内容

档案内容通常是一段描述性的文本。例如：

```markdown
姓名：小林
年龄：28
职业：全栈开发者
性格：理性、直接、偶尔幽默
特殊习惯：喜欢在代码里加注释
```

这会在 AI 的上下文中呈现，帮助 AI 根据你的身份调整语气和风格。

### 档案配置项

| 字段            | 说明                                     |
| --------------- | ---------------------------------------- |
| **名称 (Name)** | 档案的唯一标识符，也是宏 `{{user}}` 的值 |
| **显示名称**    | UI 中展示的名称                          |
| **图标**        | 显示在消息中的头像（Emoji 或图片路径）   |
| **内容**        | 档案主体内容，通过 `{{persona}}` 宏注入  |
| **启用状态**    | 禁用的档案不会出现在选择列表中           |

## 切换档案

### 全局切换

你可以在用户档案管理中选择一个「全局默认档案」——该档案会应用于所有会话。

### Agent 绑定

在 Agent 编辑器中，可以为特定 Agent 绑定一个用户档案。当使用该 Agent 时，绑定的档案会覆盖全局设置。

### 优先级

1. Agent 绑定的用户档案（最高）
2. 全局默认档案
3. 无档案（`{{user}}` 和 `{{persona}}` 宏返回空）

## 数据隔离

每个用户档案独立存储：

- 配置文件和内容
- 头像历史记录
- 独立的 Markdown 样式设置（可选择跟随 Agent 或自定义）
- 独立的正则替换规则

删除档案时，文件会被移入系统回收站。

## 消息中的用户标识

在导出的对话记录中，系统会根据消息的元数据快照显示对应的用户名和头像——即使该档案后续被删除或修改，历史消息中的标识信息不会改变。

## 与宏的配合

用户档案与宏系统深度集成：

| 宏            | 说明                   |
| ------------- | ---------------------- |
| `{{user}}`    | 当前用户档案的名称     |
| `{{persona}}` | 当前用户档案的完整内容 |

在 Agent 的预设消息中使用这些宏，可以动态根据当前用户调整系统提示词。例如：

```markdown
你是 {{char}}，一个乐于助人的 AI 助手。
你当前正在与 {{user}} 对话。
用户的背景信息：
{{persona}}
```
