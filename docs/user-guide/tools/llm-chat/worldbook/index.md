# 📖 世界书 (Worldbook) 概览

世界书 (Worldbook) 是一种**关键词触发的上下文注入机制**，灵感来自 SillyTavern 的 World Info / Lorebook 功能。它允许你预先定义大量背景设定，然后在对话中通过关键词自动匹配并注入相关内容。

## 什么是世界书？

想象你正在写一部奇幻小说——你需要记录大量设定：人物关系、地理环境、魔法体系、历史事件……世界书就是这些设定的数据库。

与传统的一整段系统提示词不同，世界书采用**“按需注入”**策略：只有当前对话中出现了相关关键词，对应的设定才会被加入上下文。这让你可以在不撑爆 Token 预算的前提下，维护一个庞大的知识库。

## 核心流程

1. **库管理**: 创建、删除、导入或导出世界书库。
2. **条目定义**: 在世界书中添加具体的设定条目（Entry），并配置其触发关键词。
3. **关联激活**: 将世界书关联到特定的 [Agent](../agents/index) 或设为全局生效。
4. **扫描匹配**: 对话过程中，系统自动扫描消息内容，匹配关键词。
5. **动态注入**: 匹配成功的条目按照预设的优先级和位置，实时组装进发送给 LLM 的提示词中。

## 快速上手

### 从零创建
1. 打开 **聊天设置** -> **世界书** -> 点击 **「世界书库管理」**。
2. 点击 **「新建世界书」**，输入名称即可。

### 从 SillyTavern 导入
系统高度兼容 SillyTavern 格式的 World Info JSON 文件：
1. 在世界书管理器中点击 **「导入」**。
2. 选择 `.json` 角色设定文件，系统会自动解析所有条目、关键词及高级元数据。
