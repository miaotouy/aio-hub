## 生成结果的自动入库

**媒体生成中心**与系统的 **资产管理器**（Asset Manager）深度集成。每次媒体生成任务成功完成后，结果会自动：

1. **导入资产系统** —— 生成的文件（图片/视频/音频）自动注册为系统资产，保存在统一存储目录下
2. **嵌入元数据** —— 提示词、模型名称、参数快照、种子值等生成信息被直接嵌入到文件元数据中（支持 PNG/JPEG/WebP/MP4 等格式）
3. **持久化衍生数据** —— 额外的生成信息（如修订后的提示词 `revisedPrompt`）作为一个独立的 JSON 文件存放，关联到资产

这意味着你**不需要手动保存**生成结果——它们已经被系统妥善管理。

## 在工作台中查看

### 右侧面板：生成历史

工作台右侧的 **生成历史** 面板展示了当前会话相关的生成结果：

- 按类型筛选（全部 / 图片 / 视频 / 音频）
- 点击缩略图即可预览（图片使用 ImageViewer，视频和音频分别使用对应的播放器）
- 点击刷新按钮手动更新列表

### 画廊标签页

切换到 **「画廊」** 标签页，可以浏览**所有**通过媒体生成中心产出的结果：

- **搜索**：按提示词或模型名称搜索（支持后端搜索和前端分页两种方式）
- **筛选**：按媒体类型（图片/视频/音频）快速过滤
- **卡片预览**：鼠标悬停时显示提示词摘要，图片直接显示缩略图，视频支持 hover 播放
- **操作按钮**（鼠标悬停浮现）：
  - 📋 **查看参数** —— 查看该资产对应的生成参数详情
  - ⬇️ **下载** —— 下载原始文件到本地
  - 🗑️ **删除** —— 将资产从媒体生成中心的来源列表中移除。**注意**：这仅解除资产与本工具的关联，不会物理删除资产文件，你仍可以在资产管理器中找到它。

> 画廊采用**无限滚动**加载，滚动到底部自动加载更多结果。

## 跨工具集成：在资产管理器中查看

媒体生成中心在系统资产管理器中注册了两个便捷功能，你可以在任何工具的资产上下文中使用：

1. **查看生成参数** —— 当你在资产管理器中选中一个由媒体生成中心产出的文件时，右键菜单中会出现「查看生成参数」选项。点击后会弹出参数详情对话框，显示：
   - 提示词（Prompt）
   - 负向提示词（Negative Prompt）
   - 使用的模型（Model ID）
   - 核心参数快照（Size, Seed, Steps, CFG Scale 等）
   - 修正后的提示词（Revised Prompt，如果模型自动优化了你的输入）

2. **元数据降级读取** —— 如果资产文件中没有内嵌元数据，系统会尝试从文件中的元数据系统提取；如果仍不存在，会提示「未找到生成参数信息」。

## 会话持久化

除了资产，**会话记录**（包括提示词、分支树、参数配置等上下文）也被自动持久化到本地文件系统：

- **存储位置**：`appDataDir / media-generator / sessions /`
- **索引文件**：`sessions-index.json` 维护所有会话的轻量级索引（ID、名称、时间戳、任务数）
- **会话文件**：每个会话一个独立的 JSON 文件，包含完整的节点树和状态

这意味着：

- 关闭应用后重新打开，所有会话历史和生成记录完好无损
- 支持创建多个会话并在它们之间切换
- 每个会话可独立修改名称方便管理

## 批量管理操作

### 会话模式下

在创作会话中，点击头部工具栏的「批量操作」按钮进入批量模式：

- **批量选中**：勾选多条消息记录
- **批量下载**：选中多条结果的下载（功能开发中，敬请期待）
- **批量删除**：确认后删除选中的消息及其关联任务

### 任务列表模式下

在「任务列表」标签页，你可以：

- **清空已结束任务**：一键移除所有已完成/失败/已取消的任务
- **单个操作**：每个任务卡片支持复制提示词、取消生成、重新生成、下载结果、删除

## 常见场景

### Q：生成的结果保存在哪里？

文件存储在系统的资产目录中（`appDataDir / assets /`），通过资产管理器统一管理。会话元数据则存储在 `appDataDir / media-generator /`。

### Q：如何找回之前生成的作品？

三种方式：

1. 在画廊中按时间倒序浏览所有结果
2. 通过会话管理器切换到历史会话，查看当时的生成流
3. 在资产管理器中按来源模块 `media-generator` 筛选

### Q：删除任务会影响资产文件吗？

**不会**。删除任务只会从任务列表中移除该记录，已经导入到资产系统的文件不会被删除。要清理资产，请使用资产管理器。

### Q：可以批量导出生成参数吗？

当前支持按分支导出 Markdown 或 JSON 格式的生成记录（包含提示词、参数快照和结果列表）。可以在会话的对话框中触发导出操作。
