## 简介

**媒体生成中心**（Media Generator）是一个基于 LLM 的媒体创作工具，允许你通过自然语言提示词直接生成图片、视频或音频内容。它采用**会话式创作**和**快速单次生成**两种工作模式，并深度集成了系统的资产管理体系——所有生成结果自动入库，随时回顾、下载或二次创作。

### 核心能力

| 能力         | 说明                                                     |
| ------------ | -------------------------------------------------------- |
| **图片生成** | 支持多种模型引擎，可配置分辨率、质量、风格、负向提示词等 |
| **视频生成** | 生成短视频素材，可配置分辨率和时长                       |
| **音频生成** | 支持 Suno / 标准音频模型，可配置质量、风格标签等         |
| **会话管理** | 分支式会话树，支持多轮迭代和版本比较                     |
| **快速模式** | 即开即用，不创建会话，适合快速试验                       |

## 进入工作台

在侧边栏点击 🎨 **媒体生成中心** 图标进入。主界面分为四个标签页：

| 标签         | 用途                                               |
| ------------ | -------------------------------------------------- |
| **工作台**   | 核心创作区：左侧参数面板、中间生成流、右侧历史画廊 |
| **画廊**     | 全屏浏览所有生成结果，支持搜索和筛选               |
| **任务列表** | 全局任务队列，查看进度、重试、下载                 |
| **生成设置** | 精细配置：自动命名、提示词优化、翻译、并发等       |

## 选择生成类型

工作台左侧的 **参数面板** 顶部分为三种媒体类型：

- 🖼️ **图片**（默认）—— 生成静态图像
- 🎬 **视频** —— 生成短视频
- 🎵 **音频** —— 生成音乐或音效

点击相应按钮即可切换。**每种类型的参数配置独立保存**，切换时不会丢失已设置的参数。

## 配置生成参数

### 选择生成模型

在参数面板的「生成模型」区域选择一个已配置好的 LLM 渠道和模型。系统会自动识别模型的能力（图片生成 / 视频生成 / 音频生成），并只显示匹配当前媒体类型的模型。

> 模型支持的具体参数范围（如分辨率选项、质量等级、风格等）由 **模型元数据系统** 驱动。如果某些参数选项为空，请前往「设置 → 模型元数据」为对应模型补充能力描述。

### 图片参数详解

| 参数                               | 说明                                                                                             |
| ---------------------------------- | ------------------------------------------------------------------------------------------------ |
| **分辨率**                         | 支持 **预设**、**自定义宽高**（64~4096px）或 **宽高比模式**（针对 xAI 等模型），支持一键交换宽高 |
| **质量级别**                       | 如标准 / HD，视模型支持情况而定                                                                  |
| **生成风格**                       | 如自然 / 艺术 / 写实等风格预设                                                                   |
| **负向提示词**                     | 描述你不希望在图片中出现的内容                                                                   |
| **背景设置**                       | 透明背景 / 纯色背景等                                                                            |
| **输入保真度**                     | 使用参考图时，控制对原始特征的保留程度                                                           |
| **流式预览图**                     | 部分模型支持在生成过程中逐步显示预览图                                                           |
| **批量生成 (n)**                   | 一次生成多张图片（1~4 张）                                                                       |
| **内容审核 / 输出格式 / 输出压缩** | 视模型支持情况                                                                                   |

### 视频参数

- **分辨率**：1280x720（720p）或 1920x1080（1080p），也支持自定义
- **时长**：5 秒或 10 秒

### 音频参数

- **Suno 模式**：灵感模式（简单描述）或自定义模式（指定风格标签、标题、纯音乐等）
- **非 Suno 模式**：标准音质或高音质

### 高级参数

点击「高级参数」折叠面板展开：

- **种子 (Seed)**：-1 为随机，指定值可复现相同结果
- **迭代步数 (Steps)**：影响生成质量和速度
- **引导系数 (CFG Scale)**：提示词 adherence 程度

## 两种创作模式

### 创作会话（默认）

适合多轮迭代和精细化创作：

1. 在输入框中输入提示词（支持 `Ctrl+Enter` 发送）
2. 系统创建一条「用户」消息和一条「助手」生成任务
3. 生成完成后，你可以：
   - **重试**：在消息上点击重试按钮，基于相同参数重新生成
   - **分支编辑**：编辑用户消息创建新分支，在不同方向上迭代
   - **切换分支**：通过分支选择器在不同结果间对比
4. 所有会话自动保存，支持创建新会话和历史切换

> **提示**：开启「连续对话」开关后，系统会将历史消息作为上下文发送给模型，支持迭代修改（如"把上一张图改成冷色调"）。此功能需要模型支持视觉输入。

### 快速生成

适合快速试验：

1. 切换到「快速生成」模式
2. 输入提示词，点击发送
3. 任务执行后显示在任务队列中，不创建会话记录
4. 支持「停止全部」和「清理已完成」操作

## 使用参考图

如果你希望基于现有图片进行图生图或变体生成：

1. **拖放**：将图片文件拖入输入框区域
2. **粘贴**：从剪贴板粘贴图片
3. **选择文件**：点击输入框工具栏的附件按钮

参考图会显示在输入框上方，可随时移除。

## 查看与管理生成结果

### 工作台右侧面板

点击右侧面板的「生成历史」，可按类型筛选（全部 / 图片 / 视频 / 音频），点击缩略图即可预览。

### 画廊标签页

切换到「画廊」标签页查看全量生成结果：

- **搜索**：按提示词或模型名称搜索
- **筛选**：按媒体类型过滤
- **预览**：鼠标悬停可查看提示词，点击打开全屏查看器
- **操作**：查看生成参数、下载到本地、从画廊移除

### 任务列表标签页

切换到「任务列表」可查看全局任务队列：

- 统计概览：总任务数、进行中、排队、完成、失败
- 筛选和搜索
- 每个任务卡片显示提示词、模型、进度或预览图
- 支持：复制提示词、取消任务、重新生成、下载、删除
