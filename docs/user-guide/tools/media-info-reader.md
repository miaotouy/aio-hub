# AI 信息解析

> 从图片中提取 AI 生成参数、工作流、角色卡等元数据，一键还原创作配方。

## 功能概览

- **Stable Diffusion WebUI 参数提取** — 自动读取图片中的正反提示词（Positive/Negative Prompt）、采样参数、模型信息、Civitai 资源引用等。
- **ComfyUI 工作流还原** — 解析图片中嵌入的 ComfyUI 工作流 JSON，可直接复制用于复现或调试。
- **SillyTavern 角色卡片可视化** — 识别并结构化展示 ST 角色卡（chara_card_v2/v3），包括角色设定、提示词策略、对话示例等。
- **AIO Bundle 解析** — 解析 AIO Agent Bundle 包（支持 ZIP 压缩包），展示 Agent 身份、思维导向、预设消息流、运行参数和关联资产。
- **完整元数据查勘** — 展示图片的原始 EXIF 全量 JSON，供高级用户深度排查。
- **一键发送到聊天** — 将解析结果按当前标签页的内容发送到 LLM 聊天工具，方便继续追问或参考。

## 快速上手

1. 在主页点击 **AI 信息解析** 卡片进入工具。
2. 将一张 AI 生成的图片（PNG / JPG / WebP）**拖放**到左侧预览区，或点击「选择图片」按钮打开文件选择器，也可以直接 **Ctrl+V** 粘贴剪贴板中的图片。
3. 右侧面板会自动解析并显示匹配的标签页，按优先级排序：
   - **AIO 可视化**（若图片包含 AIO Bundle）
   - **WebUI Info**（若图片包含 WebUI 参数）
   - **ComfyUI Workflow**（若图片包含工作流）
   - **ST 可视化**（若图片包含角色卡）
   - **完整元数据**（兜底，展示原始 EXIF）
4. 点击「发送到聊天」可将当前标签页的内容推送至 LLM 聊天界面。

## 功能详解

### 图片导入

左侧预览面板提供三种导入方式：

| 方式         | 操作                                       |
| ------------ | ------------------------------------------ |
| **拖放**     | 从文件管理器直接拖拽图片文件到预览区域     |
| **选择文件** | 点击「选择图片」按钮，从系统文件对话框选取 |
| **粘贴**     | 复制图片后按下 Ctrl+V，直接从剪贴板导入    |

图片加载后，解析器会立即启动，预览区右上角会显示「正在解析...」状态。

### WebUI 信息

当图片由 Stable Diffusion WebUI / AUTOMATIC1111 / Forge 等生成时，此标签页展示：

- **Positive Prompt** — 正面提示词全文。
- **Negative Prompt** — 负面提示词全文。
- **Generation Info** — 生成参数键值对，包括 Steps、Sampler、CFG scale、Seed、Size、Model、VAE 等。
- **Civitai Resources** — 图片引用的 Civitai 模型资源列表，按类型（Checkpoint / LoRA / LyCORIS / Embedding）使用不同颜色标签，并显示权重。

每个区块均配有复制按钮，方便将提示词或参数单独复制使用。

### ComfyUI Workflow

展示从图片中提取的 ComfyUI 工作流 JSON，以只读代码编辑器呈现，支持语法高亮和折叠。可一键复制完整工作流，用于在 ComfyUI 中导入复现。

### SillyTavern 角色卡

支持 chara_card_v2 和 chara_card_v3 格式，以可视化的卡片布局展示角色信息：

- **角色概要** — 名称、版本号、作者、标签云。
- **角色设定** — 角色描述（Description）、性格特质（Personality）。
- **提示词策略** — System Prompt、Post History Instructions。
- **对话与场景** — 场景设定（Scenario）、首条消息（First Message）、备用问候语（Alternate Greetings）、对话示例（Message Examples）。
- **作者备注** — Creator Notes。

同时提供 **ST 源码** 标签页，展示原始 JSON，供需要手动编辑或二次处理的用户使用。

### AIO Bundle 可视化

当图片中嵌入了 AIO Agent Bundle 时，以精美的信息卡片展示每个 Agent：

- **身份信息** — 头像、显示名称、版本、描述、模型 ID、资产数量。
- **思维导向** — LLM 思维规则（think rules）和 Persona / System Prompt。
- **预设消息流** — 以时间线形式展示预设消息（system / user / assistant），含角色标签、注入策略（深度注入 / 位置注入）和模型匹配条件。
- **运行参数** — 采样控制（Temperature / Top P / Reasoning Effort）和上下文压缩配置。
- **关联资产** — 按分组展示资产 ID 和大小。

支持 JSON 和 YAML 两种格式。同时提供 **AIO 源码** 标签页，可查看或复制原始 Manifest。

### 完整元数据

以代码编辑器形式展示图片的完整 EXIF JSON，包含所有可读取的元数据字段。适合高级用户进行深度排查或数据分析。

### 发送到聊天

点击预览区右上角的「发送到聊天」按钮，系统会：

1. 将当前显示的标签页内容（提示词、工作流、角色卡或元数据）格式化后发送到 LLM 聊天工具。
2. 若图片已作为资产导入，附件也会一并发送。

例如，在 WebUI Info 标签页点击发送，聊天输入框会自动填入：

```markdown
来自图片的 Stable Diffusion WebUI 信息：
\`\`\`text
Positive Prompt: ...
Negative Prompt: ...

Steps: 30, Sampler: DPM++ 2M Karras, CFG scale: 7, ...
\`\`\`
```

## 高级用法

### 在聊天中复用解析结果

解析出提示词或角色卡后，可一键发送到 LLM 聊天工具，让 AI 助手帮你：

- 优化或改写提示词
- 分析角色卡的设定
- 诊断 ComfyUI 工作流问题
- 提取角色卡中的关键设定用于创作

### 多端协同

桌面端解析的图片可通过资产管理器统一管理，方便反复查阅或跨工具调用。

## 常见问题

### 为什么有的图片解析不出内容？

并非所有图片都包含元数据。以下情况可能导致解析为空：

- 图片是通过截图或屏幕录制生成的（不包含 AI 生成参数）。
- 图片经过了有损压缩或二次保存，导致元数据被剥离。
- 图片是由不同的 AI 工具生成的，而该工具的元数据格式未被本项目支持。

此时右侧面板会显示「未检测到 AI 生成信息」，建议检查图片来源。

### 支持哪些图片格式？

支持 PNG、JPG、JPEG、WebP 四种常见格式。PNG 格式对元数据的兼容性最好，建议将重要的 AI 生成图片保存为 PNG。

### 解析的结果安全吗？

所有解析完全在本地完成，无需联网。图片文件和元数据均不会上传至任何服务器。

### 我能在手机或平板上使用吗？

移动端暂未提供此工具，请在桌面端使用。

### 解析后的数据能否导出？

当前标签页的每个信息区块都配有复制按钮。你也可以通过「发送到聊天」将结构化的内容导出到 LLM 聊天界面，再自行复制保存。
