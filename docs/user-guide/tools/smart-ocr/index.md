# 🔍 智能 OCR 快速上手

在主页点击 **智能 OCR** 卡片进入工具。

> **一句话介绍**：将图片中的文字"提取"出来，支持截图、扫描件、长图等多种场景。

智能 OCR 工具支持 **4 种识别引擎**，并内置**智能切图**功能，能自动将长图按空白区域分割后再识别，大幅提升识别效率和准确率。

---

## 基本概念

| 概念               | 说明                                           |
| ------------------ | ---------------------------------------------- |
| **OCR 引擎**       | 负责文字识别的"大脑"，支持本地、云端等多种类型 |
| **智能切图**       | 将长图自动分割成多个小块，逐个识别后再合并结果 |
| **图片块 (Block)** | 切图后产生的独立识别单元                       |
| **历史记录**       | 每次识别的结果会自动保存，方便回溯             |

---

## 第一步：选择图片

### 支持格式

- 常见图片格式：PNG、JPG、JPEG、BMP、WebP
- 支持单张和批量上传

### 上传方式

1. 点击预览区域的上传按钮
2. 直接将图片文件拖拽到预览区域
3. 支持多选文件，一次上传多张图片

> **贴士**：批量上传多张图片后，可以逐一选择处理，也可以一键批量识别全部。

---

## 第二步：执行识别

1. 在左侧**控制面板**中确认已选择好 OCR 引擎（默认为 Tesseract.js）
2. 点击底部 **"识别当前图片"** 按钮（单张识别）或 **"批量识别全部"** 按钮（多张图片时显示）

![识别按钮位置示意]（TODO: 截图 - 控制面板底部的识别按钮区域）

### 识别过程中的状态反馈

| 状态         | 说明                   |
| ------------ | ---------------------- |
| `pending`    | 等待处理               |
| `processing` | 正在识别中             |
| `success`    | 识别成功               |
| `error`      | 识别失败（可点击重试） |

识别进度会实时在右侧结果面板中更新，每个图片块的状态一目了然。

---

## 第三步：查看结果

识别完成后，右侧结果面板会展示每个图片块的识别文本：

### 结果操作

| 操作              | 说明                       |
| ----------------- | -------------------------- |
| **复制文本**      | 一键复制识别结果到剪贴板   |
| **编辑文本**      | 手动修正识别不准确的文字   |
| **忽略/取消忽略** | 标记不需要的块，复制时跳过 |
| **重试单个块**    | 对识别失败的块重新识别     |
| **重试所有失败**  | 一键重试所有失败块         |

### 结果导出

识别后的文本支持复制粘贴到任意应用。未来版本计划加入更多导出格式。

---

## 第四步：管理历史记录

每次成功的识别都会自动保存到历史记录中：

1. 点击控制面板顶部的 **历史记录** 按钮（📋 图标）
2. 在历史记录对话框中可以：
   - **加载历史**：查看之前识别的完整结果
   - **重新识别**：用当前引擎配置重新识别历史图片

## 常见问题

**Q：识别出的文字乱七八糟怎么办？**
A：检查是否选对了引擎和语言。中文图片请选择"简体中文+英文"，遇到复杂排版可以尝试切换 VLM 引擎。

**Q：长图识别结果不理想？**
A：确保智能切图功能已启用，并适当调整切图参数（见进阶配置）。

**Q：识别速度太慢？**
A：可以增加 Tesseract 引擎的并发数，或切换到 Native OCR 引擎（速度最快）。
