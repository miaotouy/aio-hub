# 工作区基础

## 概述

**工作区** 就是你在 AIO Hub 中看到的整个界面——它是你与所有工具交互的大本营。理解工作区的各个区域和操作方式，能让你在使用过程中更加得心应手。

简单来说，AIO Hub 的工作区由三个核心部分组成：

```
┌──────────────────────────────────────────────────┐
│                    标题栏                        │
├──────────┬───────────────────────────────────────┤
│          │                                       │
│  侧边栏  │            主内容区域                 │
│  (导航)  │     (工具页面 / 聊天 / 设置)          │
│          │                                       │
│          │                                       │
└──────────┴───────────────────────────────────────┘
```

- **标题栏**：顶部区域，显示当前页面名称，提供主题切换、通知、设置、窗口控制等全局操作入口。
- **侧边栏**：左侧区域，用于管理已打开的工具标签页，支持折叠和多种显示模式。
- **主内容区**：占据最大面积的中央区域，所有工具页面都在这里打开和运行。

> 💡 **小贴士**：标题栏和侧边栏的背景具有通透感。如果你开启了 **UI 模糊** 并设置了应用壁纸，它们会透出应用内部壁纸；如果你开启了 **窗口特效**（如 Windows 的 Mica 或 macOS 的模糊），它们甚至能透出你的系统桌面背景。

## 快速上手

> 跟着下面的步骤走一遍，你就能熟悉工作区的基本操作了。

### 1. 认识你的界面

启动 AIO Hub 后，你会看到这样的主界面：

![主界面标注](TODO: 截图-标注出标题栏、侧边栏和主内容区的位置)

- **左侧**一列图标就是侧边栏，显示你当前已打开的工具。
- **顶部**一条是标题栏。
- **中间**大片区域是主内容区，默认显示**主页**，你可以在这里看到所有可用的工具。

### 2. 打开一个工具

1. 在**主页**中找到你想要使用的工具卡片。
2. 点击工具卡片，该工具将会在主内容区打开。
3. 同时，该工具的图标会出现在左侧的**侧边栏**中。

![打开工具](TODO: 截图-在主页点击工具卡片打开)

### 3. 同时打开多个工具

AIO Hub 支持多标签页架构——你可以同时打开多个工具，并在侧边栏中快速切换。

1. 点击侧边栏顶部的 **🏠 主页** 图标回到主页。
2. 点击另一个工具卡片。
3. 新工具会打开，且侧边栏会增加一个新的工具图标。
4. 你可以通过点击侧边栏中的不同图标来切换工具。

![多工具切换](TODO: 截图-点击侧边栏不同图标切换已打开的工具)

### 4. 关闭工具

当某个工具用完后，可以关闭它：

- 在侧边栏中，鼠标悬停在工具名称上，右边会出现一个 **×** 按钮，点击即可关闭
- 或者在工具图标上 **右键点击**，选择「关闭标签页」

![关闭工具](TODO: 截图-鼠标悬停在侧边栏工具上，显示关闭按钮)

### 5. 进入设置

点击侧边栏底部或标题栏右侧的 **⚙️ 设置** 图标，进入设置页面。这里可以配置 LLM 服务、调整外观等。

## 功能详解

### 🟦 标题栏

标题栏是工作区的"控制中心"，所有全局操作都能在这里完成。

![标题栏区域标注](TODO: 截图-标题栏标注出各个区域)

#### 左侧：品牌与导航

- **品牌标识**：左侧显示 AIO Hub 的 Logo 和名称
- **菜单按钮**：如果你在设置中将侧边栏切换为「下拉菜单」或「抽屉」模式，左侧会出现一个菜单按钮（☰），点击即可展开导航

> 侧边栏的显示模式可以在 **设置 → 外观 → 侧边栏模式** 中调整，有三种可选：
>
> - **固定侧边栏**（默认）：侧边栏始终显示在左侧
> - **下拉菜单**：点击标题栏左侧按钮展开导航
> - **抽屉**：点击标题栏左侧按钮从左侧滑出导航

#### 中间：当前页面标题

显示当前正在使用的工具名称。如果你打开了不同工具，标题会相应变化。

#### 右侧：全局操作区

| 按钮  | 功能                | 说明                                                         |
| :---- | :------------------ | :----------------------------------------------------------- |
| 👤    | **用户档案**        | 选择不同角色与 AI 对话，每个档案有独立的名字和头像           |
| 🌓    | **主题切换**        | 快速切换浅色 / 深色 / 跟随系统。点击后下拉菜单可选           |
| 📥    | **下载管理**        | 查看和管理文件下载进度与历史（如未显示可在设置中开启）       |
| 🔔    | **消息通知**        | 查看系统推送的通知消息（如任务完成提醒）                     |
| ⚙️    | **设置**            | 进入设置页面配置各项参数                                     |
| **—** | **最小化**          | 将应用最小化到任务栏                                         |
| **□** | **最大化/还原**     | 在窗口模式和全屏之间切换                                     |
| **×** | **关闭/隐藏到托盘** | 关闭窗口。如果开启了「最小化到托盘」，点击后会隐藏到系统托盘 |

> 💡 **macOS 用户请注意**：在 macOS 上，标题栏左侧会预留空间给原生的"红绿灯"按钮（关闭/最小化/最大化），窗口控制按钮会由系统提供，因此标题栏右侧不会显示这些按钮。

#### 分离窗口的标题栏

当你把一个工具拖拽成独立窗口后，分离窗口也有自己的标题栏，但内容略有不同：

- 显示该工具的名称和图标
- 右侧多了一个 **↩ 回归主窗口** 按钮，点击即可将工具放回主窗口

### 🟩 侧边栏

侧边栏是工作区的“活动中心”，它以标签页的形式管理你当前正在使用的工具。

#### 侧边栏结构

侧边栏中从上到下依次排列：

- **🏠 主页** — 点击返回工具平铺主页，从这里打开新工具。
- **🧩 扩展** — 管理和浏览插件市场（功能还在开发中）。
- **已打开的工具** — 你当前打开的工具图标会显示在这里，带有激活状态指示。

![侧边栏拆解](TODO: 截图-在侧边栏上标注出各部分说明)

#### 操作方式

| 操作         | 怎么做                           | 效果                                     |
| :----------- | :------------------------------- | :--------------------------------------- |
| **切换工具** | 左键点击已打开的工具图标         | 主内容区切换到该工具页面                 |
| **关闭工具** | 鼠标悬停在图标上，点击右侧 **×** | 关闭该工具标签页（仅展开模式）           |
| **调整顺序** | **长按并拖动** 工具图标          | 上下移动调整工具在侧边栏的排列顺序       |
| **分离窗口** | **右键点击** → 在新窗口中打开    | 把工具拖拽为独立窗口                     |
| **分离窗口** | **按住并向右拖拽** 图标超过 40px | 快速将工具分离为独立窗口（仅限 Windows） |
| **更多操作** | **右键点击** 工具图标            | 弹出菜单：分离窗口或关闭标签页           |

> 💡 **小贴士**：侧边栏收起时，为了节省空间，关闭按钮会隐藏，此时请通过**右键菜单**来关闭工具。

### ⬜ 主内容区

主内容区是工作区中占比最大的区域，所有工具都在这里运行。

## 高级用法

### 🪟 多窗口模式

AIO Hub 最强大的特性之一就是能把任意工具**创建成独立窗口**。这在多任务处理时特别好用。

#### 触发分离的方式

AIO Hub 提供两种便捷方式将工具分离为独立窗口：

- **快捷拖拽**：在侧边栏找到工具图标，**按住鼠标左键向右拖拽**超过 40px，工具会自动“弹出”为独立窗口。
  - ⚠️ **注意**：由于系统兼容性限制，目前**快捷拖拽**主要支持 Windows 系统。macOS 用户由于系统交互冲突，该功能已被禁用，请统一使用右键菜单进行分离。
- **右键菜单**：在侧边栏工具图标上点击**右键**，选择「在新窗口中打开」。

#### 场景示例

- **一边聊天一边查资料**：将 **LLM 聊天** 分离到副屏，主窗口保持打开浏览器或文档工具。
- **同时对比两个工具**：将 **JSON 格式化** 分离，主窗口打开 **文本差异对比**，实现跨窗口的数据处理。

#### 恢复分离窗口

当需要将工具收回到主窗口时，可以使用以下方式：

- **回归按钮**：点击分离窗口标题栏右侧的 **↩ 回归主窗口** 按钮。
- **直接关闭**：直接关闭分离窗口，工具会自动回到主窗口的侧边栏中。

> 💡 所有分离窗口的位置和大小都会被自动记住，下次启动应用时会恢复到上次的位置。

## 常见问题

### Q1：我不小心把工具拖拽成独立窗口了，怎么恢复？

别担心，有几种方式可以恢复：

1. 点击分离窗口标题栏右侧的 **↩ 回归主窗口** 按钮
2. 直接关闭分离窗口，工具会自动回到主窗口的侧边栏中

### Q2：侧边栏太宽了，能缩窄一点吗？

可以。侧边栏支持**折叠模式**：

1. 点击侧边栏底部的 **◀** 按钮
2. 侧边栏会收缩为只显示图标的一窄条
3. 再次点击 **▶** 按钮即可展开

如果你想完全隐藏侧边栏，可以在 **设置 → 外观 → 侧边栏模式** 中切换为「下拉菜单」或「抽屉」模式。

### Q3：我可以自定义侧边栏中工具的顺序吗？

可以。在侧边栏中**长按**任意工具图标，然后**上下拖动**即可调整顺序。松开鼠标后位置就被记住了，下次启动应用时顺序不变。

### Q4：同时打开太多工具，主内容区标签页太多了怎么办？

- **关闭不用的工具**：在侧边栏右键工具图标，点击菜单中的 `关闭标签页`。在展开模式下可以直接点击标签上的 **X** 关闭。

### Q5：为什么我的标题栏/侧边栏没有毛玻璃效果？

AIO Hub 的“模糊”分为两种，请根据你的需求检查设置：

#### 情况 A：你想透出“应用内的壁纸” (UI 模糊)

这种效果是在应用内部实现的，你需要：

1. 在 **外观 → 壁纸管理** 中开启壁纸，并选一张图。
2. 在 **外观 → 界面质感** 中开启「启用 UI 特效」和「UI 模糊」。
3. 调低「UI 基础不透明度」滑块。
   _注：如果不设置应用壁纸，UI 模糊将没有背景可滤，看起来就是纯色。_

#### 情况 B：你想透出“系统桌面背景” (窗口特效)

这种效果由操作系统提供，具有更原生的质感：

1. 在 **外观 → 窗口特效** 中开启开关。
2. 在「窗口背景特效」下拉框中选择 **模糊 (Blur)**、**亚克力 (Acrylic)** 或 **云母 (Mica)**。
3. 开启「窗口背景不透明度」并调低滑块。
4. **关键点**：在「界面质感」中将 **壁纸不透明度** 调低（甚至设为 0），否则应用内的壁纸会挡住窗口背后的画面。
   _注：此功能为实验性，且在 Linux 系统上暂不可用。_
