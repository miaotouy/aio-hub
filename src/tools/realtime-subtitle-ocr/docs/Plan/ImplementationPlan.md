# 实时字幕OCR (Realtime Subtitle OCR) 交互与布局重构计划书

本文档详细规划了“实时字幕OCR”前端模块的交互与布局重构步骤，旨在解决AI对交互理解偏差的问题，打造极简、直观、符合专业工作流的字幕工具。

> 修订时间：2026-07-05
> 修订人：Gugu_Kilo (雪鸮妹妹)
> 状态：Implementing (重构中)

---

## 1. 重构背景与核心痛点

在上一轮重构中，虽然解决了配置持久化和内存安全问题，但交互设计依然存在严重硬伤：

1. **设置面板冗余**：监控设置单独占用右侧一个大面板，导致空间极度浪费，且干扰核心工作流。
2. **按钮分布混乱、重复**：
   - 监控框控制按钮（打开、聚焦、开始/停止）散落在顶部工具栏，与预览画面割裂。
   - 字幕处理按钮（复制全部、导出 SRT）在顶部工具栏和下方时间轴区域重复出现，界面臃肿。
3. **预览区视觉死板**：预览区采用死黑背景（`#000`），在暗色/亮色主题下都显得极其突兀，破坏了 AIO Hub 的通透感和毛玻璃美学。
4. **比例失调**：预览区和编辑区比例不合理，无法满足“大画面监控、小区域编辑”的实际视频/直播监控场景。

---

## 2. 交互重构方案：极简与逻辑聚合

我们将彻底重构界面，将控制、设置、预览和编辑逻辑进行物理与视觉上的重新聚合：

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 顶部工具栏 (Toolbar): [标题] [状态] | [采样频率] [去重灵敏度] [OCR引擎] [云配置]  │
├──────────────────────────────────────────────────────────────────────────┤
│                                     │                                    │
│                                     │                                    │
│          监控预览与控制区            │            当前字幕编辑            │
│           (Live Preview)            │           (Subtitle Edit)          │
│                70%                  │                 30%                │
│                                     │                                    │
│                                     │                                    │
├─────────────────────────────────────┴────────────────────────────────────┤
│                               拖拽分割条                                  │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│                              字幕时间轴列表                               │
│                 [复制全部] [导出 SRT] [一键清空] (聚合于此)                 │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### 2.1. 核心改动设计

#### A. 监控设置移入顶部栏 (`rsocr-toolbar`)

- 彻底移除右侧独立的 `MonitorConfig.vue` 组件。
- 将核心监控设置项（采样频率、去重灵敏度、OCR 引擎）以紧凑、精致的表单件形式**平铺在顶部工具栏右侧**。
- **引擎额外配置气泡化**：在 OCR 引擎选择框右侧放置一个设置小按钮（如齿轮图标），点击后通过 `el-popover`（气泡卡片）展示当前引擎的额外配置（如云端 OCR 的配置选择、Tesseract 的语言选择等），保持顶部栏的极简与整洁。
- 顶部工具栏不再保留任何字幕处理按钮（如复制、导出、清空）。

#### B. 监控框控制按钮合并至预览区 (`LivePreview.vue`)

- 将与监控框有关的按钮（`打开监控框`、`聚焦监控框`、`开始/停止监控`）全部移入 `LivePreview.vue`。
- 按钮将作为预览区底部的控制条（Control Bar）存在，与画面预览、延迟/aHash 信息紧密结合。
- 预览区不再使用死黑背景（`#000`），改用符合主题外观系统的通透背景 `var(--input-bg)`，并配合毛玻璃效果，使其在未就绪时也显得精致优雅。

#### C. 预览区与编辑区 7:3 比例分栏

- 主体上方区域由三栏简化为两栏：
  - **左侧（70% 宽度）**：`LivePreview.vue`（大画面监控预览 + 核心控制按钮）。
  - **右侧（30% 宽度）**：当前字幕大字编辑框（`editor-panel`）。
- 这样既保证了监控画面的清晰度，又给字幕编辑留出了合理的输入空间。

#### D. 字幕处理按钮去重与聚合

- 顶部工具栏不再保留“复制全部”、“导出 SRT”和“一键清空”按钮。
- 这些按钮统一聚合在下方 `SubtitleTimeline.vue`（字幕时间轴列表）的头部工具栏中，实现“结果处理”逻辑的完美闭环。

## 3. 实施步骤规划

### 阶段一：顶部工具栏与配置项重构 (`RealtimeSubtitleOcr.vue`)

1. 在 `RealtimeSubtitleOcr.vue` 的顶部工具栏中，平铺引入：
   - 采样频率：使用紧凑的 `el-slider`（限制宽度，如 120px）或 `el-select`。
   - 去重灵敏度：使用 `el-select`。
   - OCR 引擎：使用 `el-select`。
   - 引擎额外配置：在引擎选择框右侧放置一个齿轮按钮，点击弹出 `el-popover`。
     - 当引擎为 `cloud` 时，在 Popover 中展示云端 OCR 配置选择。
     - 当引擎为 `tesseract` 时，在 Popover 中展示语言选择。
2. 移除顶部工具栏中的字幕处理按钮。
3. 移除右侧 `MonitorConfig.vue` 的挂载。

### 阶段二：预览区与控制按钮合并 (`LivePreview.vue`)

1. 重构 `LivePreview.vue`，接收监控状态、控制函数和窗口状态作为 props/emits：
   - `isRunning`
   - `isMonitorBoxDetached`
   - `monitorRect`
   - `lastFrameUrl`
   - `lastHash`
   - `latency`
2. 在 `LivePreview.vue` 底部新增控制条，放置：
   - `打开监控框` / `聚焦监控框` 按钮。
   - `开始监控` / `停止监控` 按钮。
3. 优化 `.live-preview__screen` 样式，将 `background: #000` 改为 `background: var(--input-bg)`，并添加毛玻璃和通透边框。

### 阶段三：布局与比例调整

1. 修改 `RealtimeSubtitleOcr.vue` 中的主体上方布局：
   - 预览区容器宽度设为 `flex: 7;` 或 `width: 70%;`。
   - 编辑区容器宽度设为 `flex: 3;` 或 `width: 30%;`。
2. 确保在窄屏下有合理的 `min-width` 限制，防止挤压变形。

### 阶段四：时间轴与去重按钮完善 (`SubtitleTimeline.vue`)

1. 在 `SubtitleTimeline.vue` 的头部工具栏中，保留并美化：
   - `复制全部` 按钮。
   - `导出 SRT` 按钮。
   - 新增 `一键清空` 按钮（从主入口移入，并绑定二次确认逻辑）。
2. 确保所有按钮样式与项目规范一致。

## 4. 验证与测试标准

1. **视觉通透性**：切换明暗主题，验证预览区背景是否通透，无死黑。
2. **比例正确性**：验证预览区是否占 70% 宽度，编辑区是否占 30% 宽度。
3. **功能完整性**：
   - 验证在顶部栏修改采样率、引擎等配置是否能实时生效并持久化。
   - 验证在预览区点击开始/停止、打开/聚焦监控框是否正常工作。
   - 验证时间轴区域的复制、导出、清空功能是否正常。
