画板 (Sketch Pad) ​

概述 ​

画板 是 AIO Hub 内置的轻量绘图工具，支持自由手绘和矢量形状编辑。你可以用它来快速草绘想法、标注截图、制作简单的示意图，或者直接将画好的内容发送给 AI 对话。

核心特性：

  • 混合图层系统：位图手绘 + 矢量形状，自由叠加
  • 多种画笔：铅笔、马克笔、橡皮擦，支持压感平滑
  • 矢量形状：矩形、椭圆、线段、箭头、文字、图片
  • 图层管理：无限图层，支持排序、锁定、透明度调节
  • 项目管理：自动保存、缩略图预览、导入/导出
  • 跨工具协作：一键将画布内容发送到 AI 对话

快速上手 ​

1. 创建新画布 ​

打开画板工具后，你会看到项目画廊界面。页面左上角显示标题"草图画板"，右上角有操作按钮。点击右上角的 「新建草图」 按钮：

  1. 输入草图名称（可选，默认自动生成）
  2. 选择画布比例：超宽 (2.39:1)、宽屏 (1.85:1)、HD (16:9)、传统 (4:3)、方形 (1:1)、竖屏 (9:16) 或自定义
  3. 选择自定义比例时可以手动输入宽高（100~8192 px）
  4. 可选是否创建背景图层，以及背景色（白色、透明或自定义颜色）
  5. 点击「创建」

画布创建后会自动进入编辑器。

💡 右上角还有「导入草图 (.aiosk)」按钮，可以导入之前导出的项目文件。

2. 开始绘制 ​

编辑器界面由画布和三个悬浮面板组成：

┌──────────────────────────────────────────────────────┐
│ [← 返回]   [ 工具按钮组（居中） ]   [保存/导出/发送] │  ← 悬浮工具栏
├──────────────────────────────────────────────────────┤
│                                                      │
│                                                      │
│                      画布区域                        │
│                                                      │
│                                                      │
│ ┌──────────┐                           ┌───────────┐ │
│ │ 属性面板 │                           │ 图层面板  │ │
│ └──────────┘                           └───────────┘ │
└──────────────────────────────────────────────────────┘
  • 工具栏（顶部悬浮，分三段）：
    • 左侧：返回画廊按钮
    • 中间：工具选择 + 撤销/重做 + 重置视图
    • 右侧：保存、导出（下拉菜单）、发送到对话
  • 属性面板（左下角悬浮）：显示当前工具或选中对象的属性，可折叠
  • 图层面板（右下角悬浮）：管理图层，默认折叠为一个图标按钮，点击展开

选择铅笔工具（快捷键 B），在画布上按住鼠标拖动即可开始绘制。

💡 窗口较窄时，工具栏会自动将部分工具收入「更多」菜单中，保持界面整洁。

3. 保存与导出 ​

  • Ctrl+S：保存项目（下次打开画板时可以继续编辑）
  • 导出按钮：将画布导出为 PNG / JPG / WebP 图片文件
  • 发送到对话：将当前画布内容直接发送到 AI 聊天作为附件

工具详解 ​

通用工具 ​

工具图标快捷键说明
选择V选中并移动/变换矢量对象
抓手H拖拽平移画布视图（也可按住空格键临时切换）

画笔工具（位图绘制） ​

画笔工具在位图图层上绘制，产生像素级的笔画。

工具图标快捷键说明
铅笔B细腻的手绘线条，适合勾线和书写
马克笔M较粗的半透明笔触，适合标注和涂色
橡皮擦E擦除位图图层上的像素

属性面板中可以调整：

  • 大小：笔刷粗细
  • 颜色：绘制颜色
  • 透明度：笔触的不透明度

形状工具（矢量对象） ​

形状工具在对象图层上创建矢量元素，创建后可以随时选中并修改属性。

工具图标快捷键说明
矩形R拖拽绘制矩形，支持圆角
椭圆O拖拽绘制椭圆/圆形
线段L绘制直线
箭头A绘制带箭头的线段
文字T点击放置文本框，双击编辑内容
图片I从文件导入图片到画布

属性面板中可以调整：

  • 描边颜色/宽度：边框样式
  • 填充颜色：内部填充
  • 圆角（矩形）：圆角半径
  • 虚线样式：实线或虚线

选中对象后的操作 ​

用选择工具（V）点击矢量对象后：

  • 拖拽：移动对象位置
  • 拖拽角点：缩放对象
  • 拖拽旋转手柄：旋转对象
  • Shift+点击：多选对象
  • 框选：在空白区域拖拽画出选择框，框内对象自动选中
  • Shift+框选：追加选择（不清除已有选中）
  • Delete / Backspace：删除选中对象
  • Ctrl+A：全选当前图层的所有对象

选中后，属性面板会显示该对象的专属属性（颜色、大小、字体等），可以直接修改。

图层系统 ​

画板支持三种图层类型，可以自由混合叠加：

图层类型用途面板图标
填充图层纯色或透明的背景底色
位图图层承载画笔绘制的像素内容
对象图层承载矢量形状（矩形、文字等）

图层面板操作 ​

图层面板默认折叠为右下角的 图标按钮，点击即可展开。

  • 新建图层：面板头部右侧有三个小按钮 ，分别对应填充/位图/对象图层，点击即可创建
  • 切换活跃图层：点击图层列表中的某一层
  • 调整顺序：hover 图层条目时右侧出现 箭头按钮，点击调整顺序
  • 可见性：点击图层条目左侧的 图标切换显示/隐藏
  • 锁定：点击 图标防止误操作
  • 展开对象列表：对象图层右侧有 展开按钮，可以查看和选中图层内的各个对象
  • 删除：面板底部操作栏的 按钮（至少保留一个图层）

高级图层操作（面板底部操作栏） ​

  • 向下合并：将当前图层与下方图层合并为一层
  • 栅格化：将对象图层转换为位图图层（不可逆，但支持撤销）

智能图层切换 ​

当你选择的工具与当前活跃图层类型不匹配时，系统会自动切换到合适的图层：

  • 选择铅笔/马克笔/橡皮擦 → 自动切换到最近的位图图层
  • 选择矩形/椭圆/线段等 → 自动切换到最近的对象图层
  • 如果没有合适的图层，系统会自动创建一个

项目管理 ​

项目画廊 ​

画板的首页是项目画廊，以网格形式展示所有已保存的草图缩略图。每张卡片显示缩略图、名称、修改时间和尺寸。

  • 打开项目：点击缩略图卡片
  • 重命名：鼠标悬停在卡片上，名称右侧会出现 按钮，点击弹出重命名对话框
  • 删除：鼠标悬停在卡片上，名称右侧会出现 按钮，点击后确认删除
  • 导入：点击右上角 「导入草图 (.aiosk)」按钮，选择文件
  • 设置：点击右上角 按钮打开画板全局设置
  • 刷新索引：点击右上角 按钮同步项目列表

自动保存 ​

画板支持自动保存功能（默认每 30 秒检查一次）。当检测到未保存的修改时，系统会在后台静默保存，无需手动操作。

你可以在画板设置中调整自动保存的间隔时间，或关闭此功能。

导入与导出 ​

格式说明
.aiosk画板项目文件，包含所有图层和资产，可完整恢复编辑
PNG无损位图导出（2x 分辨率）
JPG有损压缩导出，文件更小
WebP现代格式，兼顾质量和体积

增量保存：创建当前项目的一份副本，原项目保持不变。适合在大改之前"存个档"。

发送到 AI 对话 ​

画板与 智能体对话 工具深度集成。点击工具栏右侧的「发送到对话」按钮：

  1. 当前画布会以 2x 分辨率导出为 PNG
  2. 图片自动注册为系统资产
  3. 自动跳转到 AI 对话界面，图片已添加到附件列表
  4. 你可以直接向 AI 描述画面内容或提出修改建议

这个功能非常适合：

  • 画一个 UI 草图，让 AI 帮你生成代码
  • 标注截图中的问题区域，让 AI 分析
  • 绘制流程图或示意图，让 AI 帮你完善

画板设置 ​

点击画廊界面右上角的 按钮可以打开画板全局设置：

  • 新建项目默认值：默认画布尺寸、是否自动创建填充图层/对象图层
  • 画笔默认属性：默认颜色、大小
  • 形状默认属性：默认描边宽度、填充颜色
  • 文字默认属性：默认字号、字体
  • 行为设置：自动保存开关和间隔时间

常见问题 ​

Q1：画笔画不出东西？ ​

检查以下几点：

  1. 当前活跃图层是否为位图图层（画笔只能在位图图层上绘制）
  2. 当前图层是否被锁定（锁定的图层无法编辑）
  3. 当前图层是否可见（隐藏的图层也无法绘制）
  4. 画笔颜色是否与背景色相同

Q2：形状工具拖拽后没有出现形状？ ​

确认当前活跃图层是对象图层。如果系统没有自动切换，可以手动在图层面板中选择一个对象图层，或新建一个。

Q3：撤销次数有限制吗？ ​

默认保留 80 步历史记录。由于位图操作需要存储完整的像素数据，过多的历史记录会占用较多内存。

Q4：导入的图片去哪了？ ​

通过图片工具（I）导入的图片会作为矢量对象放置在当前对象图层上。你可以用选择工具（V）选中它，然后拖拽移动或缩放。

图片文件本身由系统的资产管理器统一管理，支持去重存储。

Q5：.aiosk 文件是什么？ ​

这是画板的项目文件格式，本质上是一个 ZIP 压缩包，内含所有图层数据和引用的图片资产。你可以用它在不同设备间迁移画板项目，或分享给其他 AIO Hub 用户。

Q6：如何在画布上添加文字？ ​

  1. 选择文字工具（快捷键 T）
  2. 在画布上点击想要放置文字的位置
  3. 输入文字内容
  4. 在属性面板中调整字号、字体、颜色、对齐方式等

双击已有的文字对象可以重新编辑内容。
